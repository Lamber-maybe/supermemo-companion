The Window submenu on the main menu provides the following options (only at Middle or higher levels):

Window menu items

[edit | edit source]

Themes

[edit | edit source]
  • Themes - choose a theme that makes SuperMemo look best for you (see the video at the bottom of this page)

Layout

[edit | edit source]
  • Layout - management of layouts that determine where SuperMemo windows and toolbars are placed:
    • Layout manager - open a layout manager that makes it possible to select, apply, save or delete layouts
    • Apply default layout (Ctrl+F5) - restore the windows layout last saved with Layout : Save as default (below)
    • Warrior layout (F5) - arrange windows in the way optimized for the use of incremental reading. Compare: File : Level : Warrior layout
    • Classic layout - arrange windows in the manner known from early versions of SuperMemo, i.e. with the Statistics window conveniently aligned to the left of the element window. This arrangement is useful when you make repetitions and like to constantly follow the progress of learning. You can make this layout permanent with Layout : Save as default (below)
    • Save custom layout - save the current windows layout under a selected name
    • Save as default (Shift+Ctrl+F5) - save the current layout of windows as your default layout. Each time you enter the currently used collection, this layout will be used
    • Background color - set the color of the background window displayed with Window : Background (above). Setting the background color is the fastest way of differentiating between collections (e.g. main collection, working copy, backup, test collection, family member's copy, etc.)
    • Background picture - select a picture displayed in the SuperMemo background (if set with Window : Background above)
    • About picture - select a picture displayed in the SuperMemo splash screen as well as in Help : About

Dark mode

[edit | edit source]
  • Dark mode - sets a dark theme, and a dark stylesheet (no effect on template colors)

Icons

[edit | edit source]
  • Icons - adjust toolbar and menu icon size for different screen resolutions
    • Large - use the largest practical icon size for different screen resolutions
    • Small - use compact icons (large toolbars switch to medium or smaller bars switch to small)
    • Tiny - use the smallest practical icons for low-resolution screens

Hints

[edit | edit source]
  • Hints - turn on or turn off displaying hints when the mouse pauses over buttons. In addition, this option affects whether the status bar shows the hints when the mouse pauses over a menu option

Contents

[edit | edit source]
  • Contents (Alt+C) - select the current element in the Contents window

Status bar

[edit | edit source]
  • Status bar - show or hide the status bar (at the bottom of the screen)

Background

[edit | edit source]
  • Background (Ctrl+Alt+F10) - show or hide the background behind windows used by SuperMemo (to minimize distraction caused by background applications)

Statistics

[edit | edit source]
  • Statistics - show the window with the main learning process statistics in the currently opened collection

Element data

[edit | edit source]
  • Element data - show the window with the statistics of the currently displayed element

Ancestors

[edit | edit source]
  • Ancestors (Shift+Ctrl+X) - open the Ancestors window that makes it possible to go to a given ancestor with a click

Hide SuperMemo

[edit | edit source]
  • Hide SuperMemo (Shift+Ctrl+G) - hide SuperMemo from view and move its icon from the taskbar to the notification area near the system clock (notification area is colloquially known as the system tray). This option is particularly useful if you use Toolkit : Plan to organize your day. This way you can hide SuperMemo from view and focus on activity at hand. SuperMemo will then automatically reappear towards the end of the slot. If you pause your mouse over the SuperMemo icon in the notification area, you will be able to see how much time is left until the end of the activity when the alarm is raised.

Custom layout list

[edit | edit source]
  • 1-9 - choose one of up to nine custom layouts defined earlier and saved with Save custom layout. The selected layout automatically becomes the default layout
