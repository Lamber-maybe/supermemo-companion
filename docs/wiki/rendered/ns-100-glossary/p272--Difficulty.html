Glossary:Difficulty

difficulty
estimation of an item's difficulty in SuperMemo. The five main measures of difficulty are:
  1. Diff (SM-19 Difficulty): A dynamic, continuous value ranging from 0 (easiest) to 1 (hardest), optimized via Algorithm SM-19.
  2. SM-20 Dificulty: A dynamic, continuous value ranging from 0 (easiest) to 1 (hardest), optimized via machine learning using the Binary Win Metric (Win - R).
  3. FSRS Difficulty: Used in FSRS mode, ranging from 1.0 (easiest) to 10.0 (hardest); mapped to the 0–1 range for consistent display in SuperMemo.
  4. A-Factor (SM-15): The classic scheduling multiplier ranging from 1.2 (hardest) to 6.9+ (easiest), representing the item's bounded stabilization factor.
  5. Old Heuristic: A static 0 to 1 value estimating difficulty using five raw database statistics (first grade, repetitions, lapses, interval, and A-Factor).
