Help Us Improve Our Documentation and Videos!

If you’re struggling to find information on this website, need an overview of SuperMemo features, or would like a new help video, please consider one of these options:

  • Recommended: Add your questions or suggestions directly to relevant pages in SuperMemo Help (click the Edit tab to start writing; requires login).
  • Submit your ideas or feature requests at SuperMemopedia.
  • Request a new help video or send feedback via e-mail (e.g., using Help : E-mail : Bug report).

Your input helps us create clearer documentation and better help videos, reducing support costs and enhancing your experience with SuperMemo.

Thank you for your contributions!
