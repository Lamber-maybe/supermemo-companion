Registration

This text contains basic information for users who:

  • use a demo/trial version of SuperMemo
  • have their SuperMemo locked
  • plan to register by sending a registration e-mail
  • have sent their registration e-mail already
  • are running out of use credit
  • want to reinstall SuperMemo on a new computer

I want to become a user

If you would like to become a user of SuperMemo, you have many freeware and commercial version options:

  • Download
  • SuperMemo 19 for Windows (newest SuperMemo)

New purchase of SuperMemo

If you have just purchased your SuperMemo, send your registration data. You will receive an unlock password within a business day:

  • Hello
  • Unlock SuperMemo

Owners of SuperMemo

If you have already purchased SuperMemo before, and it is locked on a new computer or after a re-installation, you do not need to pay again.

You can do any of the following:

  • (either) open any of your collections used on the same computer. This should unlock SuperMemo
  • (or) use File : Installation : Input password (Ctrl+Shift+I) to insert the unlock password, which you received upon registration
  • (or) write to shop2026(@)supermemo(.)org to have your unlock password resent
  • (or) resubmit your registration to install SuperMemo on a new computer

Questions

If you have questions about SuperMemo, it is best to place them at SuperMemopedia.com.

Questions about the registration process can be found here.

To post your questions:

  1. Go to https://supermemopedia.com
  2. Click "Log in" (top right corner)
  3. Type Username: "SuperMemoUser"
  4. Type Password: "SuperMemo2022"
  5. Click the "Log in" button
  6. Click Ask Your Question (on the left)

You can also send questions via e-mail.
