Contents toolbar appears at the top of the Contents window.

Contents toolbar buttons

[edit | edit source]
  • Menu (Shift+F10) - open the contents menu otherwise available with a right-click over the Contents window
  • Branch processing menu - open a subset processing menu with operations on the currently selected branch (same as Process branch> on the contents menu available with a right-click)
  • Element align - align the Contents window with the element window for convenient review of elements in the Contents window (note that Synchronize button must be down for the element window to be updated as the selection moves in the Contents window)
  • Contents align - open a second Contents window and align both windows for convenient drag-and-drop operations
  • Find string (Ctrl+F) - search for a string in the currently selected branch
  • View (Shift+Enter) - display the currently selected element in the element window (needed only if Synchronize is off)
  • Collapse branch (Ctrl+Alt+Enter) - collapse the parent of the currently selected element (to easily fold the whole knowledge tree displayed in the Contents window)
  • Postpone - delay outstanding repetitions in the selected branch
  • Save - save knowledge tree nodes to a text file (optionally, the entire tree can be expanded and branch statistics can be included as well)
  • Set root - make the currently selected node as the root of the currently selected concept group
  • Set hook - make the currently selected node the hook of the currently selected concept group. This way all elements added with Add new (Alt+A), available at the bottom of the element window, will be children of the selected node
  • Info - display node statistics of the currently selected element
  • Synchronize - display elements in the element window upon changing the selection in the Contents window (see: unselecting this option in case of trouble)
  • Select current - select the element that is currently displayed in the element window (needed only if Synchronize is off)
