Toolkit : Mercy : Criteria dialog box can be used to determine the criteria by which items rescheduled with Toolkit : Mercy should be sorted.

If you have just returned from vacation or had a longer break in learning, you will probably want to reschedule outstanding repetitions over a longer period of time. This way you can avoid piles of work accumulated in SuperMemo.

Depending on the nature of the material stored in your collection, you might want to use completely different criteria. For example, if you have added lots of important job-related information shortly before leaving for vacation, you might want to quickly go through this recently introduced material first. In other cases, you might want to ease the stress related to the return to repetitions by picking the easiest material first. You might also want to minimize the damage to the long-term learning process by starting with elements in which you have invested most, or those whose repetitions have been delayed to the largest degree.

Use the following dialog box to set weight to individual criteria from 0% to 100%:

Sorting criteria

[edit | edit source]
  1. Priority - importance of elements as determined by priority (as well as some other criteria, including the forgetting index)
  2. Lateness - degree of delay of repetitions. Usually short-interval elements are more likely to be forgotten as a result of delay. Use this criterion to minimize the forgetting rate during repetitions of the rescheduled material
  3. Investment - investment of your time in memorizing items. The investment is determined by the number of repetitions, the number of memory lapses, and most of all, the length of the interval. Use this criterion to protect the material you have worked hardest on. Even though this is the material that is least likely to be forgotten, each memory lapse of high-investment material will result in a most painful increase in workload
  4. Easiness - easiness of the elements as determined by the interval and the number of memory lapses (the fewer the lapses and the longer the interval, the easier the element). Use this criterion to maximize the speed of learning, and to minimize the stress of handling the outstanding material. You can opt for maximizing Easiness at times of crisis (i.e. when you cannot devote sufficient attention to repetitions). This way you can quickly dislodge most of the outstanding material without adding many repetitions of newly forgotten elements. You can deal with the hardest part only then when the catch-up hardship period comes to the end
  5. Recency - how recently the elements have been introduced into the learning process. The recency is determined by (1) intervals (shorter intervals indicate recency), (2) memory lapses (lapses refute recency) and (3) number of repetitions (recent items have fewer repetitions made). Use this criterion when the most recently introduced material is of highest recall priority (e.g. in cases you memorized some critical job-related material before leaving for vacation). This criterion is also very useful if you are going through a crisis with your repetitions (Do I really need to spend that much time with SuperMemo?). Recency criterion will help you quickly realize what great stuff you have learned recently and what a pity it would be to lose it to forgetting. If the break is long enough, you will also see what a painful proportion of elements you have already lost!

Warning! By abusing Recency and Easiness criteria you can indefinitely reschedule some harder long-interval elements! You might be very disappointed in a year or two to discover that some elements have gained very long intervals without a single repetition and have entirely faded from your memory!

The percentage values of the impact of individual criteria on sorting can be seen to the right from the slide-bars. Change those values by moving the sliders. For example, if you want to repeat rescheduled material with the easiest material coming first, put all sliders to the left and the Easiness slider on the right (Easiness=100%).

Note that Investment criterion cannot drop below 2%. This is only to prevent the situation in which all criteria have the value 0%.

Sorting options

[edit | edit source]
  1. Use sorting criteria specified above - sorting will proceed along the criteria determined by the sliders: priority, lateness, investment, etc.
  2. Reverse sorting order - sorting will be reversed. This can be used for example to use the Easiness criterion to sort repetitions starting from the most difficult material
  3. Retain current repetition sequence - when using Spread in the browser: reschedule without sorting, i.e. without changing the current sorting of elements (i.e. the same repetition sequence will be executed but all repetitions will take place later than originally planned)
  4. Randomize - randomize the sequence of rescheduled repetitions. This criterion might be used if you frequently run short-term rescheduling (e.g. to equalize the number of repetitions per day). Paradoxically, randomizing the sequence of rescheduled repetitions may show minimum side effects. Randomization prevents biased rescheduling that may accumulate over time when frequently using Mercy with the same set of fixed criteria such as easy-items-first, recent-items-first, etc.

Quickly rescheduling outstanding material using a given set of criteria

[edit | edit source]
  1. Choose Toolkit : Mercy (e.g. by pressing Shift+Alt+M)
  2. Choose Criteria and set the sliders according to your preferences
  3. Choose the maximum acceptable number of repetitions per day and type it in at Number of elements per day. Alternatively, choose the period in which all outstanding repetitions should be done and type it in at Rescheduling period
  4. Choose Update, e.g. by pressing Enter. This will display the rescheduling parameters: number of items per day, length of the rescheduling period, the date on which last outstanding repetitions will be made, etc.
  5. If you are satisfied with your selection, click OK. Otherwise type in new parameters and choose Update again

工具包 :慈悲 ：标准对话框可用于确定标准项目重新安排工具包 :慈悲应该排序。

如果您刚刚度假回来或学习休息时间较长，您可能需要重新安排时间突出的重复在较长一段时间内。这样你就可以避免在 SuperMemo 中积累大量的工作。

根据您存储的材料的性质收藏，您可能想使用完全不同的标准。例如，如果您在出发度假前不久添加了许多与工作相关的重要信息，您可能需要首先快速浏览一下最近介绍的材料。在其他情况下，您可能希望通过首先选择最简单的材料来缓解与返回重复相关的压力。您可能还想通过以下方式开始最大程度地减少对长期学习过程的损害：元素您投入最多的项目，或者重复被延迟最大程度的项目。

使用以下对话框将各个标准的权重设置为 0% 到 100%：

排序标准

  1. 优先级- 重要性元素由下式确定优先级（以及其他一些标准，包括遗忘指数）
  2. 迟到- 重复的延迟程度。通常很短-间隔 元素更容易因为延误而被遗忘。使用此标准可以最大限度地减少重复重新安排的材料期间的遗忘率
  3. 投资- 在记忆上投入时间项目。投资由重复次数、次数决定记忆衰退，最重要的是，长度间隔。使用此标准来保护您最努力制作的材料。尽管这是最不可能被遗忘的材料，但每个记忆衰退高投资材料将导致工作量的最痛苦的增加
  4. 轻松- 的容易程度元素由间隔和数量记忆衰退（失误越少，间隔越长，元素就越容易）。使用此标准可以最大限度地提高学习速度，并最大限度地减少处理问题的压力优秀的材料。您可以选择最大化轻松在危机时刻（即当你无法对重复投入足够的注意力时）。这样你就可以快速去除大部分优秀的材料不添加许多新遗忘的重复元素。只有当追赶困难期结束时，你才能应对最困难的部分
  5. 新近度- 最近多久元素已被引入到学习过程中。新近度由 (1) 确定间隔（较短的间隔表示新近），(2)记忆衰退（失误反驳最近的情况）和（3）重复次数（最近的情况）项目重复次数较少）。当最近介绍的材料具有最高回忆优先级时（例如，如果您在休假之前记住了一些与工作相关的重要材料），请使用此标准。如果你正经历重复的危机，这个标准也非常有用（我真的需要花那么多时间在 SuperMemo 上吗？）。新近度标准将帮助你快速认识到你最近学到了什么很棒的东西，以及如果忘记了它会是多么遗憾。如果休息的时间足够长，你也会看到有多么痛苦的比例元素你已经输了！

警告！通过滥用新近度和轻松您可以无限期地重新安排一些更难的长期标准间隔 元素！一两年后你可能会非常失望地发现有些元素已经收获了很长的时间间隔没有一次重复，并且已经完全从你的记忆中消失了！

各个标准对排序的影响的百分比值可以从滑动条的右侧看到。通过移动滑块来更改这些值。例如，如果您想重复重新安排的材料，并先将最简单的材料放在前面，请将所有滑块放在左侧，然后将滑块放在左侧。轻松右侧滑块（轻松=100%）。

请注意投资标准不能低于2%。这只是为了防止所有标准的值为 0% 的情况。

排序选项

[编辑|编辑源代码]
  1. 使用上面指定的排序标准- 排序将按照滑块确定的标准进行：优先级、迟到、投资等
  2. 逆序排列- 排序将被颠倒。例如，这可以用于使用轻松从最难的材料开始对重复进行排序的标准
  3. 保留当前重复序列- 使用时传播在浏览器：重新安排而不排序，即不改变当前的排序元素（即，将执行相同的重复序列，但所有重复将比原计划晚进行）
  4. 随机化- 随机化重新安排的重复的顺序。如果您经常进行短期重新安排（例如，为了均衡每天的重复次数），则可以使用此标准。矛盾的是，随机化重新安排的重复顺序可能会显示出最小的副作用。随机化可以防止有偏差的重新安排，这种重新安排在频繁使用时可能会随着时间的推移而累积慈悲具有相同的固定标准，例如简单项目优先、最近项目优先等。
