Intro

[edit | edit source]

If these are your first hours with SuperMemo 20, this page includes some startup information that you might find useful.

Download

[edit | edit source]
  • you can download SuperMemo or SuperMemo collections at the time of purchase
  • in addition, in response to your order, you will receive an automatic e-mail message that will include a download link. Your product will be available there indefinitely (unless removed from the Store)
  • if you lost the link, and you need to download or re-download, please write to receive assistance

Figure: The Download Now button available at purchase or when following the download link sent via e-mail. If you fail to download while ordering, you can always download later. Please write for help

Installation

[edit | edit source]
  • run the downloaded file (sm20inst.exe) to install SuperMemo. Note that newer Windows may complain of lack of digital signature marking the file as "Unknown Publisher". You can ignore that warning by choosing More info, and Run anyway (in Windows)
  • if you know of a version update, and SuperMemo still downloads the old version, it is a result of caching. Try downloading in InPrivate/incognito window
  • if you have not installed SuperMemo yet, see installation instructions

Password

[edit | edit source]
  • please use File : Installation : Input password, or use Ctrl+i, to insert the unlock password. New users receive their password via e-mail within a business day of registration
  • you are not likely to need the password again (unless you install on more than two computers). You can always get it from us via e-mail on request
  • if you cannot provide the password because File : Installation : Input password is inactive, your SuperMemo must have already been unlocked (or your version is compatible with the prior database format and requires no upgrade)
  • if you get a message "Wrong password" please check if you are using SuperMemo 20 (i.e. not SuperMemo 2008, SuperMemo 15, or SuperMemo 19 Beta or Gamma, etc.). Choose Help : About in SuperMemo to see which version you are using. If you are still in trouble, please do not hesitate to write

For more details see: Unlocking SuperMemo

Upgrading collections

[edit | edit source]
  • Very old collections: you may need to download SuperMemo 16 to upgrade older collections. SuperMemo 16 is free for users of SuperMemo 17 and later
  • Preserving Contents: see if your categories upgraded correctly to concepts (e.g. to avoid pitfalls). Check if the root and hook settings are correct. Avoid using the main element as the root in large/complex collections
  • Efficient learning: if you have a mature learning process going on in your collection, use F7 and Compute to compute accurate DSR model data (that makes the new algorithm run most effectively)
  • Learning performance: please have a peek at Toolkit : Statistics : Analysis : Use [top] : Efficiency [bottom] : R-Metric [bottom] from time to time. As of SuperMemo 20, you do not need to worry about negative R-metric. It just shows that Algorithm SM-19 might still be converging faster than the combined power of Algorithm Arena

升级收藏

[编辑|编辑源代码]
  • 非常古老的收藏：你可能需要下载 SuperMemo 16升级旧收藏。 SuperMemo 16 对 SuperMemo 17 及更高版本的用户免费
  • 保存内容：看看你的类别正确升级到概念（例如，为了避免陷阱）。检查是否根和钩子设置正确。避免使用主元素作为大型/复杂的根收藏品
  • 高效学习：如果你有一个成熟的学习过程收藏, 使用F7和计算来计算准确的DSR型号数据（这使得新算法运行最有效）
  • 学习表现：请看一下工具包 :统计数据 :分析 :使用[上]：效率 [下]：R 公制 [下]时不时地。从 SuperMemo 20 开始，您无需担心负 R 度量。它只是表明算法 SM-19 的收敛速度可能仍然比算法的总和更快算法竞技场
