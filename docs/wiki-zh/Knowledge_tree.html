Knowledge tree is a tree structure in which all elements of a SuperMemo collection are organized. The knowledge tree is presented in the contents window (Contents). Particular nodes of the tree can hold up to a thousand children each, but for performance reasons it is recommended to keep no more than a hundred children elements in a single node. Some authors use the term knowledge hierarchy to refer to the knowledge tree. For more, see: Knowledge tree in SuperMemo
