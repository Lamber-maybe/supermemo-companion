Introduction

[edit | edit source]

SuperMemo features a number of various windows (element window, Contents, Statistics, etc.) and toolbars (Read, Edit, Tools, etc.) as well as number of interface elements (e.g. status bar, background, etc.) which you can show/hide, position and/or resize to organize your SuperMemo learning space to suit your preferences depending the available monitor and screen resolution as well as how well you know the program and what kind of information you are interested in.

Figure: SuperMemo for Windows on the Professional level (File : Level : Professional). Collection statistics are displayed on the left. Element's ancestors at the mid bottom show the place of the article in the Biochemistry branch. Element's statistics at shown at the bottom-right. Element window in the center of the screen displays a topic on glycolysis in cancer (a result of a split executed on the Glycolysis article imported from Wikipedia). Note R-Metric of 17.351% which shows a huge advantage of Algorithm SM-18 over Algorithm SM-15 (on that particular day of learning)

You can rely on one of the predefined SuperMemo layouts as well as create any number of your own.

Layouts are stored in the BIN\layouts.ini file. This file also can be modified manually if have backed it up safely beforehand.

SuperMemo interface parts that are saved in a layout

[edit | edit source]
  • Element window - size and position
  • Contents window - visibility, size and position
  • Statistics window - visibility, size and position
  • Element data window - visibility, size and position. Additionally, you can expand/collapse select parameter groups to economize space
  • Ancestors window - visibility, size and position
  • Status bar - visibility
  • Background - show or hide the background behind windows used by SuperMemo in order to minimize distraction caused by background applications. The background can either be a solid color (Window : Layout : Background color) or a custom picture (Window : Layout : Background picture). In Windows 10, the background can also be used to differentiate between collections (e.g. when using Alt+Tab)

Figure: In Windows 10 or later, SuperMemo background can be used as a nice visual collection chooser between active collections opened in separate instances of SuperMemo running simultaneously. Use Alt+Tab to switch between collections, or click the collection of your choice.

Saving a custom layout

[edit | edit source]

After you have arranged your SuperMemo desktop:

  1. choose Window : Layout : Save custom layout
  2. Enter the name of the layout in the New Layout dialog box,
  3. Click OK

Additionally, you can save your layout as default. The default layout is a layout which is automatically applied when you run SuperMemo, or restored to when you close the Contents window or element browser (see: automatic layouts. To save a layout as default, select Window : Layout : Save as default (Shift+Ctrl+F5).

Applying a custom layout

[edit | edit source]

Setting the default layout

[edit | edit source]
  1. press Ctrl+F5, or
  2. choose Window : Layout : Apply default layout from the main menu, or
  3. Right-click the Layout manager button in the navigation bar, and select Apply default layout, or
  4. Click the Apply default button button on the Tools toolbar (if visible)

Choosing from the custom layout list

[edit | edit source]

Select Window : 1..9 to choose one from up to nine custom layouts recently defined and saved. Note that the selected layout automatically becomes the default one.

Applying from the Layout manager

[edit | edit source]
  1. click the Layout manager button in the navigation bar at the top of the element window
  2. select a given layout
  3. press Apply
  4. press Close

Layout manager

[edit | edit source]

Layout manager is a simple tool you can use to apply a selected layout (esp. if you have saved more than 9 in total), save the current position of windows and toolbars as a new layout, set a selected layout as the default one (i.e. the layout which is automatically applied when you run SuperMemo), delete layouts you no longer need, etc.

You can open Layout manager with Window : Layout : Layout manager from the main menu, or by clicking the Layout manager button on the navigation bar in the element window (note that you need to be at the Middle or higher level for these options to become available).

The Layout manager is comprised of the following elements:

  • Layout list
  • Action buttons
    • Apply - apply the selected layout
    • Save - save the current layout of windows and toolbars for future use
    • Default - make the selected layout the default one
    • Delete - delete the selected layout
    • Close - close the manager
  • Status pane, at the bottom of the window, where you can inspect which layout is the current default one and which is presently applied.

Available layouts

[edit | edit source]

Predefined layouts

[edit | edit source]

SuperMemo comes with two predefined layouts: Classic and Warrior.

You can use them as a good starting point for your own custom layout. After you have arranged everything on the screen, select Window : Layout : Save custom layout or Save as default.

You can also make either of them permanent each time you open SuperMemo with Window : Layout : Save as default (Shift+Ctrl+F5).

Classic layout

[edit | edit source]

The Classic layout is known from early versions of SuperMemo. It conveniently aligns the Statistics window to the left of the element window and Element data window. This arrangement is useful when you make repetitions and like to constantly follow the progress of learning.

Warrior layout

[edit | edit source]

The Warrior layout has been optimized for incremental reading. Not only does it arrange windows and toolbars on your screen, it also switches SuperMemo to the Professional level. You can apply it with one of the following ways:

  • press F5
  • choose File : Level : Warrior layout from the main menu
  • choose Window : Layout : Warrior layout from the main menu
  • select Warrior in the Layout manager, and click Apply, and close the manager

Automatic layouts

[edit | edit source]

In addition to the two aforementioned predefined layouts, SuperMemo automatically re-arranges the current layout whenever you switch to the Contents window (e.g. by clicking Contents in the element window) or open the element browser (e.g. via the View menu). In this new automatic layout, the left half of the screen is occupied by either the Contents window or element browser, while the other half is split in two grids: the top is filled with the element window while the Element data window is attached to the bottom. This layout makes it very easy to inspect individual elements (along with their learning parameters) from the Contents window or element browser. Closing the Contents window/element browser restores the default layout. Alternatively, you can press Esc to the do the same.

简介

[编辑|编辑源代码]

SuperMemo 具有许多不同的窗口（元素窗口,内容,统计数据等）和工具栏（阅读,编辑,工具等）以及界面元素的数量（例如状态栏、背景等），您可以显示/隐藏、定位和/或调整大小来组织您的 SuperMemo 学习空间，以满足您的喜好，具体取决于可用的显示器和屏幕分辨率以及您对程序的了解程度以及您感兴趣的信息类型。

布局存储在BIN\layouts.ini文件。如果事先安全备份了该文件，也可以手动修改。

布局管理器

[编辑|编辑源代码]

布局管理器是一个简单的工具，您可以使用它来应用选定的布局（尤其是如果您总共保存了超过 9 个布局）、将窗口和工具栏的当前位置保存为新布局、将选定的布局设置为默认布局（即运行 SuperMemo 时自动应用的布局）、删除不再需要的布局等。

你可以打开布局管理器与窗户 :布局 ：布局管理器从主菜单，或单击布局管理器元素窗口中导航栏上的按钮（请注意，您需要位于中或更高级别才能使这些选项可用）。

布局管理器由以下元素组成：

  • 布局列表
  • 操作按钮
    • 申请- 应用选定的布局
    • 保存- 保存窗口和工具栏的当前布局以供将来使用
    • 默认- 将选定的布局设置为默认布局
    • 删除- 删除选定的布局
    • 关闭- 关闭管理器
  • 状态窗格位于窗口底部，您可以在其中检查哪种布局是当前默认布局以及当前应用的布局。

预定义布局

[编辑|编辑源代码]

SuperMemo 有两个预定义布局:经典和战士。

您可以将它们用作您自己的自定义布局的良好起点。安排完屏幕上的所有内容后，选择窗户 :布局 ：保存自定义布局或保存为默认值。

您还可以在每次打开 SuperMemo 时将其中任何一个设为永久窗户 :布局 : 保存为默认值（Shift+Ctrl+F5）。

经典布局

的经典布局是从 SuperMemo 的早期版本中得知的。它可以方便地对齐统计数据窗户的左侧元素窗口和元素数据窗户。当您进行重复并喜欢不断跟踪学习进度时，这种安排非常有用。

武士布局

的战士布局已优化增量阅读。它不仅可以在屏幕上排列窗口和工具栏，还可以将 SuperMemo 切换到专业 水平。您可以通过以下方式之一应用它：

  • 按F5

  • 选择战士在布局管理器中，然后单击申请，并关闭管理器

自动布局

[编辑|编辑源代码]

除了前面提到的两个预定义布局，每当您切换到 SuperMemo 时，SuperMemo 都会自动重新排列当前布局内容窗户（例如，通过单击内容在元素窗口）或打开元素浏览器（例如通过查看菜单）。在这个新的自动布局中，屏幕的左半部分被内容窗户或元素浏览器，而另一半则分为两个网格：顶部充满元素窗口而元素数据窗户附在底部。这种布局使得检查各个元素（及其学习参数）变得非常容易内容窗户或元素浏览器。关闭内容窗户/元素浏览器恢复默认布局。或者，您可以按ESC做同样的事情。
