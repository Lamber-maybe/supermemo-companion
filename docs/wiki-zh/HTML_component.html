HTML is the default text component in SuperMemo. It is based on editing and rendering engines of Internet Explorer. As such it carries all the rendering power and compatibility range of this powerful Microsoft product. This also means that all bugs and shortcomings should also be familiar. The good news is that, in Windows 10, most of the old HTLM problems are gone.

As all necessary Internet Explorer components are part of Windows, you do not need to worry about installing Internet Explorer. You can use any web browser while using SuperMemo.

There are 3 ways in which you are most likely to add HTML text to your collection (Middle or higher level):

  1. Notes: if you would like to type in some notes that will later be processed with incremental reading, use Edit : Add a note (Alt+N). For example, after hearing a piece of science news on TV, you may want to quickly type in a note to SuperMemo. Press Alt+N and type in the message.
  2. Articles: if you find an interesting article on the net, select it in your web browser and copy it to the clipboard (e.g. with Ctrl+C). You can then paste the article stored in the clipboard to SuperMemo with Ctrl+N (Edit : Add a new article). Article is similar to Note. The difference is that Article will paste the text from the clipboard to your newly created element, whereas Note will create a blank new element.
  3. Multiple articles: if you have a couple of articles opened in a web browser, use Edit : Web import (Shift+F8). For more details, see Web import.

Figure: A topic with an article about henipavirus in the HTML component on the left and 4 images taken from the article with Download images (Ctrl+F8) (available on the component context menu), which downloaded, localized, and tiled them in their individual image components on the right. These 4 images will propagate to all extracts and cloze deletions ultimately produced from this article.

If you want to import an HTML file to an existing HTML component, right-click the HTML component, and choose File : Import file. Pick the file in the dialog box and choose OK. Your HTML file will be displayed in the HTML component. Import file does not import files linked to by the imported file. However, you can import an entire website located on your hard disk or local network if you use Links : External file on the HTML component menu. In such a case, the HTML file will remain in its original location and retain all relative links.

If you would like to edit an HTML text in your favorite HTML editor, press Ctrl+F9 (or choose File : Edit file from the HTML component menu) and your HTML code will open in the HTML editor registered for editing HTML files in Windows. To preview an article in your web browser, Ctrl+click the HTML component or press F9 (or choose File : View file from the HTML component menu).

Inserting hyperlinks

[edit | edit source]

The most useful hyperlink will connect keywords with other elements in SuperMemo. To create inter-element hyperlink in HTML:

  1. display the target element in SuperMemo
  2. copy its number to the clipboard (e.g. by using Ctrl+G)
  3. select the linking keyword in the linking element
  4. paste the element number with Shift+Ins (note that Ctrl+V would paste the actual element number)

To insert hyperlinks that would point to other elements, registry texts, HTML files, URL addresses on the Internet, registry images, etc., read about the option Insert HTML link (HTML component menu).

The HTML component can interpret JavaScript. If you know some JavaScript commands, you can enhance your SuperMemo presentation with scripted HTML components.

HTML 是默认值文本组件在 SuperMemo 中。它基于 Internet Explorer 的编辑和渲染引擎。因此，它具有这款强大的 Microsoft 产品的所有渲染能力和兼容性范围。这也意味着所有的错误和缺点也应该是熟悉的。好消息是，在 Windows 10 中，大多数旧的 HTLM 问题都消失了。

由于所有必需的 Internet Explorer 组件都是 Windows 的一部分，因此您无需担心安装 Internet Explorer。使用 SuperMemo 时您可以使用任何网络浏览器。

您最有可能通过 3 种方式将 HTML 文本添加到您的收藏（中或更高水平）：

  1. 注释：如果您想输入一些稍后将被处理的注释增量阅读, 使用编辑 :添加注释（Alt+N）。例如，在电视上听到一段科学新闻后，您可能想快速在 SuperMemo 中输入一条注释。新闻Alt+N并输入消息。
  2. 文章：如果您在网上发现一篇有趣的文章，请在网络浏览器中选择它并将其复制到剪贴板（例如使用Ctrl+C）。然后，您可以将剪贴板中存储的文章粘贴到 SuperMemo 中：Ctrl+N（编辑 :添加新文章）。文章类似于注意事项。不同之处在于文章会将文本从剪贴板粘贴到您新创建的元素，而注意事项将创建一个空白新元素。
  3. 多篇文章：如果您有几篇文章在网络浏览器, 使用编辑 :网页导入（Shift+F8）。有关更多详细信息，请参阅网页导入。

图：一个主题有一篇关于亨尼帕病毒属在左侧的 HTML 组件中，以及从文章中获取的 4 个图像下载图片(Ctrl+F8)（可在组件上下文菜单），下载、本地化并将它们平铺到各自的图像中组件在右边。这 4 个图像将传播到所有提取物和完形填空删除最终由本文产生。

如果要将 HTML 文件导入现有 HTML 组件，请右键单击该 HTML 组件，然后选择文件 :导入文件。在对话框中选择文件并选择好的。您的 HTML 文件将显示在 HTML 组件中。导入文件不导入由导入文件链接的文件。但是，如果您使用，则可以导入位于硬盘或本地网络上的整个网站友情链接 ：外部文件在 HTML 上组件菜单。在这种情况下，HTML 文件将保留在其原始位置并保留所有相关链接。

如果您想在您喜爱的 HTML 编辑器中编辑 HTML 文本，请按Ctrl+F9（或选择文件 ：编辑文件从 HTML组件菜单）并且您的 HTML 代码将在注册用于在 Windows 中编辑 HTML 文件的 HTML 编辑器中打开。要在网络浏览器中预览文章，Ctrl+单击HTML 组件或按F9（或选择文件 ：查看文件从 HTML组件菜单）。

插入超链接

[编辑|编辑源代码]

最有用的超链接将关键字与其他关键字连接起来元素在 SuperMemo 中。要在 HTML 中创建元素间超链接：

  1. 在 SuperMemo 中显示目标元素
  2. 将其编号复制到剪贴板（例如，通过使用Ctrl+G）
  3. 在链接元素中选择链接关键字
  4. 粘贴元素编号Shift+Ins（请注意Ctrl+V将粘贴实际的元素编号）

插入指向其他内容的超链接元素、注册表文本、HTML 文件、Internet 上的 URL 地址、注册表图像等，请阅读有关该选项的信息插入 HTML 链接（HTML组件菜单）。

HTML 组件可以解释 JavaScript。如果您了解一些 JavaScript 命令，则可以使用脚本化 HTML 组件来增强 SuperMemo 演示文稿。
