Installation

[edit | edit source]

To install SuperMemo on your computer do the following:

Order and download

[edit | edit source]
  1. Order and download SuperMemo 20:
    1. Go to the SuperMemo Store home page
    2. Click Add to cart corresponding to SuperMemo 20 for Windows (this will add SuperMemo 20 to your shopping cart)
    3. Click Check out
    4. Click the large PayPal button at the Express checkout section at the top of the page. This will open a PayPal pop-up box which you will use to log into your PayPal account. Once you are in, review the displayed information, and click the Continue to Review Order at the bottom of the pop-up window. This will redirect you to the Payment page back in SuperMemo Store
    5. Click Pay now
    6. Once your order has been processed and you file(s) prepared, click Download Now under SuperMemo 20
    7. In the Save As dialog box, in the Navigation pane along the left side of the dialog box, click the Desktop button, then click Save to download SuperMemo 20 Install Wizard (i.e. sm20inst.exe) onto the desktop

Installation

[edit | edit source]
  1. Run the downloaded installer file sm20inst.exe to install and run SuperMemo 20
  2. Click the "Click here to register SuperMemo" red bar
  3. Fill out and send the registration form (video guide)
  4. Unlock SuperMemo with your unlock password sent in response to your registration (within a business day)

Uninstallation

[edit | edit source]

In 5 seconds (brute force)

[edit | edit source]

To uninstall, delete the folder in which you installed SuperMemo.

Remember that this will likely delete everything related to SuperMemo: collections, plan schedules, sleep timeline, etc. Deleting the folder manually may still leave 2-3 entries in the Windows registry (if you let SuperMemo run in administrator mode).

In 30 seconds (smart)

[edit | edit source]

You can uninstall SuperMemo (without removing your learning material) if you follow these steps:

  1. Run Control Panel
  2. In the Control Panel window, choose Uninstall program (in older Windows Uninstall or change a program)
  3. In the Uninstall or change a program window, scroll down to SuperMemo, and choose Uninstall

If you cannot find SuperMemo entry in the Control Panel, use sm20unin.exe located in the SuperMemo folder (C:\SuperMemo by default).

The SuperMemo Uninstall program was designed to remove only the files added during installation. It should not delete your collections, plan schedules, sleep timeline, etc. If you want them removed, you will need to do it manually in your favorite file manager.

卸载

[编辑|编辑源代码]

5秒内（暴力）

[编辑|编辑源代码]

要卸载，请删除安装 SuperMemo 的文件夹。

请记住，这可能会删除与 SuperMemo 相关的所有内容：收藏品、计划时间表、睡眠时间线等。手动删除文件夹可能仍会在 Windows 注册表中留下 2-3 个条目（如果您让 SuperMemo 在管理员模式下运行）。

30秒内（智能）

[编辑|编辑源代码]

如果您按照以下步骤操作，则可以卸载 SuperMemo（无需删除您的学习材料）：

  1. 运行控制面板
  2. 在控制面板窗口，选择卸载程序（在较旧的 Windows 中卸载或更改程序）
  3. 在卸载或更改程序窗口，向下滚动到超级备忘录，并选择卸载

如果你找不到超级备忘录在控制面板中输入，使用sm20unin.exe位于 SuperMemo 文件夹中（C:\SuperMemo默认情况下）。

SuperMemo 卸载程序旨在仅删除安装过程中添加的文件。它不应该删除你的收藏品、计划时间表、睡眠时间线等。如果您希望删除它们，则需要在您最喜欢的文件管理器中手动执行此操作。
