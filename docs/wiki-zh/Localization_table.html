Translating the interface of SuperMemo

[edit | edit source]

With Toolkit : Options : Language : Localization table, you can quickly translate the entire interface of SuperMemo to your native language (only at the Beginner and Basic levels).

The whole translation can be done in 5 minutes, as the entire interface of basic SuperMemo consists of only about 50 strings!

This is the procedure:

  1. Choose Toolkit : Options : Language
  2. Click the field Localization language (or press Alt+I)
  3. Type the name of your language (e.g. Spanish)
  4. Choose Localization table
  5. In the column Translation, type in the translation of texts displayed in the column Original
  6. If necessary, select a localization font by clicking the Font button in the left-bottom corner of the localization table
  7. Choose OK and your translation will appear in the interface of the program

The translation file is a text file. It is kept at <SuperMemo folder>\TRANSLAT\<language>.tra

Changing the interface

[edit | edit source]

Important! Localization affects only the options available on the Beginner and Basic levels

To change the interface in SuperMemo using a downloaded interface file do the following:

  1. Download an interface file from International Interfaces for SuperMemo (only 1000 bytes!)
  2. Copy the interface file to the subfolder TRANSLAT in the <SuperMemo> folder (the filename is <language name>.tra)
  3. Choose the interface's name in Toolkit : Options : Language : Localization language

An alternative approach is to:

  1. Download an interface from International Interfaces for SuperMemo (only 1000 bytes!)
  2. Choose Toolkit : Options : Language : Localization table
  3. Type the language name in the Localization language field (you do not have to use the name of the downloaded file)
  4. Choose Import in Localization table and select the downloaded interface file (the filename is <interface name>.tra)

翻译 SuperMemo 的界面

[编辑|编辑源代码]

与工具包 :选项 :语言 ：本地化表，您可以快速将 SuperMemo 的整个界面翻译成您的母语（仅在初学者和基础 级别）。

整个翻译可以在 5 分钟内完成，因为基本 SuperMemo 的整个界面仅由大约 50 个字符串组成！

这是程序：

  1. 选择工具包 :选项 :语言
  2. 单击该字段本地化语言（或按阿尔特+我）
  3. 输入您的语言名称（例如西班牙语）
  4. 选择本地化表
  5. 在专栏中翻译，输入该列中显示的文本的翻译原创
  6. 如有必要，通过单击选择本地化字体字体本地化表左下角的按钮
  7. 选择好的您的翻译将出现在程序界面中

翻译文件是文本文件。它被保存在<SuperMemo 文件夹>\TRANSLAT\<语言>.tra
