Toolkit : Options on the main menu opens the options dialog that makes it possible to customize SuperMemo and the learning process. See: Quick overview (video)

Options dialog includes the following tabs:

  • Learning - options affecting the learning process such as: speed of learning, working hours, Algorithm SM-18 reporting, etc.
  • Access - options affecting access to data: CD/DVD path, difficulty level, collection password, etc.
  • Language - localization options such as: language to which the current collection is translated, SuperMemo interface language, strings used in translating the interface, etc.
  • SuperMemo - other options such as: knowledge tree's branch children limit, background image, sound feedback, etc.
  • Mouse - interpretation of mouse behavior. For example, how components are edited or translated with the mouse, etc.
  • Leeches - options that help you eliminate wrongly formulated items from the learning process. If you choose Wizard, all ill-structured items will open a leech manager dialog that will help you choose optimum processing steps to eliminate problems with difficult items
  • Fonts - fonts used in SuperMemo interface, in questions, in answers, in the Contents window, etc.
  • E-mail - options for sending email from SuperMemo (e.g. to forward elements)

Global options of SuperMemo are stored in the file supermemo.ini in the SuperMemo subfolder BIN.
Options that affect only the current collection are stored in the file collection.ini in the collection subfolder (named in the same way as the collection). Toolkit : Options will modify both these files and change relevant settings

工具包 ：选项于主菜单打开选项对话框，可以自定义 SuperMemo 和学习过程。参见：快速概览（视频）

选项对话框包含以下选项卡：

  • 学习- 影响学习过程的选项，例如：学习速度, 工作时间,算法 SM-18报告等
  • 访问- 影响数据访问的选项：CD/DVD 路径、难度级别,收藏密码等
  • 语言- 本地化选项，例如：当前语言收藏被翻译成，SuperMemo 界面语言、翻译界面时使用的字符串等。
  • 超级备忘录- 其他选项，例如：知识树的分行 儿童限制、背景图像、声音反馈等。
  • 鼠标- 鼠标行为的解释。例如，如何组件用鼠标等进行编辑或翻译。
  • 水蛭- 帮助您消除错误配方的选项项目从学习过程来看。如果你选择向导, 全部结构不良项目将开设一个水蛭管理器对话框这将帮助您选择最佳的加工步骤，以消除困难的问题项目
  • 字体- SuperMemo 界面、问题、答案、答案中使用的字体内容窗户等
  • 电子邮件- 从 SuperMemo 发送电子邮件的选项（例如转发元素）

SuperMemo的全局选项存储在文件中超级备忘录.ini在 SuperMemo 子文件夹中宾格。
仅影响当前的选项收藏都存储在文件中集合.ini在收藏子文件夹（命名方式与收藏）。工具包 :选项将修改这两个文件并更改相关设置
