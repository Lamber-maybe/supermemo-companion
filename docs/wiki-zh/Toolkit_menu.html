The Toolkit submenu on the main menu provides the following options (assuming File : Level : Professional checked):

Toolkit menu items

[edit | edit source]

Commander

[edit | edit source]
  • Commander (Ctrl+Enter) - choose a SuperMemo command to execute

Calendar

[edit | edit source]
  • Calendar (Ctrl+W) - the daily and the monthly calendar of repetitions.

Sleep Chart

[edit | edit source]
  • Sleep Chart (F12) - record of repetition and/or sleep blocks that will help you see your timing pattern in learning and sleeping

Plan

[edit | edit source]
  • Plan (Ctrl+P) - a schedule manager that will help you plan your time by adjusting proportions of time you spend on individual activities

Mercy

[edit | edit source]
  • Mercy (Shift+Alt+M) - a dialog box that makes it possible to reschedule repetitions (e.g. before or after a holiday break). Use Mercy only if you want to redistribute the outstanding material in a selected period of time. If you cannot cope with material overload, you should rather use Postpone

Tasklist

[edit | edit source]
  • Tasklist (F4) - a tasklist manager to sort or edit the current tasklist

Statistics

[edit | edit source]
  • Statistics
    • Statistics - a window with the main statistics of the learning process
    • Element data - a window with the statistics related to the current element. This window also displays the repetition statistics after a repetition
    • Analysis (Shift+Alt+A) - a dialog box with various statistics related to the learning process and the SuperMemo Algorithm
    • Simulation - a dialog box that makes it possible to simulate the course of learning over many years using real data collected during repetitions (see: Theoretical aspects of SuperMemo). Please note that for a low value of the forgetting index, the simulation procedure may be inaccurate (see: FAQ). The simulation uses the OF matrix used by Algorithm SM-15 (i.e. it is not based on Algorithm SM-17)
    • Report - generate a text file with statistics displayed in the Statistics window
    • Reset data
      • Forgetting index record - reset the record of repetition grades used in computing the measured forgetting index
      • Forgetting index vs priority record - reset the record of the forgetting index measured for different priority categories (as visible in Toolkit : Statistics : Analysis : Graphs : Forgetting Index vs. Priority)
      • First grade vs A-Factor record - reset the record of the correlation between between first grades and A-Factors
      • Grade vs forgetting index - reset the correlation between the grades and the forgetting index
      • Average response time - reset the record of the average time it takes to respond to questions in your collection
      • Repetition statistics - recompute repetition counts in Workload to account for elements that have been deleted since
      • Repetitions history calendar - recompute the history of use of elements on individual days (accessible from Workload). Those statistics are kept as subsets in \history collection subfolder
      • Repetition history - delete the history of repetitions without affecting the learning process
      • Optimization matrices - reset the optimization matrices used in the learning process (RF, OF, repetition cases, forgetting curves, etc.)

Memory

[edit | edit source]
  • Memory
    • 4D Graphs (F7) - visual presentation of memory functions, their approximations as used by SuperMemo Algorithm, and options for resetting and re-computing memory data on the basis of repetition histories
    • Memory status (F2) - show changes to memory status over time
    • Total knowledge - compute the progression timeline of total knowledge
    • Export data - export memory data for all items
    • Re-compute difficulties - update item difficulties in element data for more accurate display and browsing

Neural analysis

[edit | edit source]
  • Neural analysis - build a neural queue and show how activation spreads in the network

Randomize

[edit | edit source]
  • Randomize
    • Drill - randomize the final drill queue
    • Pending - randomize the pending queue

Random test

[edit | edit source]
  • Random test - run random repetitions on various subsets of the current collection (random repetitions do not count in the learning process). The following tests can be found on the Random test submenu:
    • Resume test
    • All (Alt+F11) - run a random test on all elements
    • Pending - run a random test on pending elements
    • Memorized - run a random test on memorized elements
    • Dismissed - run a random test on dismissed elements
    • Topics - run a random test on topics
    • Items - run a random test on items
    • Filter - use the element filter and run a random test on a subset of filtered elements
    • Leeches - run a random test on difficult elements
    • Subset - run a random test on a selected element subset
    • Branch - run a random test on the currently selected branch in the Contents window

Random review

[edit | edit source]
  • Random review (F11) - randomly review elements in the current collection

Options

[edit | edit source]
  • Options - open a tabbed dialog box with options such as the default interface font, default forgetting index, path to files on an external drive, default stylesheet, password, sound feedback, background image, translation language, etc.

统计数据

[编辑|编辑源代码]
  • 统计数据
    • 统计数据- 一个包含学习过程主要统计数据的窗口
    • 元素数据- 一个窗口，其中包含与当前相关的统计信息元素。此窗口还显示重复统计信息重复
    • 分析（Shift+Alt+A) - 一个对话框，其中包含与学习过程和学习相关的各种统计数据SuperMemo 算法
    • 模拟- 一个对话框，可以使用重复期间收集的真实数据来模拟多年的学习过程（请参阅：SuperMemo 的理论方面）。请注意，对于较低的值遗忘指数，模拟过程可能不准确（参见：常见问题解答）。模拟使用算法 SM-15 使用的 OF 矩阵（即它不是基于算法 SM-17）
    • 报告- 生成一个文本文件，其中显示统计信息统计数据窗户
    • 重置数据
      • 忘记索引记录- 重置用于计算重复成绩的记录测量遗忘指数
      • 忘记索引与优先记录- 重置记录遗忘指数针对不同优先级类别进行测量（如工具包 :统计数据 :分析 :图表 :遗忘索引与优先级）
      • 一年级 vs A-Factor 记录- 重置一年级和一年级之间的相关性记录A 因素
      • 成绩与遗忘指数- 重置之间的相关性成绩和遗忘指数
      • 平均响应时间- 重置回答问题的平均时间记录收藏
      • 重复统计- 重新计算重复计数工作量考虑自此以来已被删除的元素
      • 重复历史日历- 重新计算各个日期元素的使用历史（可从工作量）。这些统计数据作为子集保存在\历史集合子文件夹
      • 重复历史- 删除重复历史记录而不影响学习过程
      • 优化矩阵- 重置学习过程中使用的优化矩阵（RF、OF、重复案例、遗忘曲线等）
