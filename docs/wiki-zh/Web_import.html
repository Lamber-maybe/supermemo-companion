Introduction

[edit | edit source]

Web import allows of massive imports of knowledge from the web. If your curiosity makes you open a hundred tabs, you can convert your browser mess into well-organized materials for learning in seconds.

If you combine web imports with incremental reading, you will never need to limit your explorations on the web. You will never suffer from a productivity decline that affects most people active on social media.

You can divide your fun with learning into (1) exploration and (2) consumption. You can explore at will within available time, and then transfer all the results to SuperMemo. At later time, with incremental reading, you can process all the imported knowledge to maximize learning benefits.

Limitations

[edit | edit source]

For 23 years, SuperMemo relied heavily on Internet Explorer. That entire legacy had to be transferred to new browsers in a short two years. If there are essential gaps in the implementation, they will be fixed as early as technically possible in proportion to demand.

Collecting import tabs

[edit | edit source]

To open the Web import dialog box and start collecting available Internet resources (e.g. pages, pictures, YouTube videos, etc.), press Shift+F8 or choose Edit : Web import from the main menu.

Previewing import list

[edit | edit source]

To open the browser tab associated with a given row in the import list, double-click it. This is one of the useful functions that are available in Edge and was not there in Internet Explorer.

To preview the plain text contents of a browser tab, click on its title or address (in the import list), and use the text preview box on the right to inspect it.

When the web page you are trying to import has already been imported to the collection, it will be denoted with a red D icon (D for Duplicate). To view the imported element in SuperMemo, Ctrl+double click the corresponding row (marked with D).

Similar blue D icon indicates tabs that are opened more than once in the browser(s).

Editing import list

[edit | edit source]

To delete a row from the import dialog and close the associated tab in the web browser, press Del.

To delete an element that has already been imported from the same address (marked with red D icon), press Ctrl+Del.

Import modes

[edit | edit source]

Import modes:

  • HTML: in this default mode, the original web page will be pulled down from the net, filtered and imported to SuperMemo (for pages that require password, Plain text needs to be used)
  • Selection: imports the selected portion of the text from the web browser. If formatted selection can be retrieved, the selection will be formatted as in the browser. In some cases, SuperMemo may need to choose between the formatted version and plain text due to dynamically inserted content. When unsure, SuperMemo will import both versions of the selection
  • Plain text: choose the Text mode to import the plain text; this mode can also be used in emergencies, e.g. when you lose the Internet connection
  • Plain text selection: select a portion of the text in the web browser, and choose the Text mode in imports to import plain text selections (e.g. portions of richly formatted lengthy articles). If the selection cannot be retrieved, the entire text will be imported (you will be able to see the fact in text preview). This happens when the text container does not support all text selection functionality
  • Link: import only a link to the document (for incremental imports when needed)
  • Parsed document (Parse): import an HTML document without the non-essential parts (headers, footers, advertising, etc.). Selections in the text are ignored
  • Plain text PDF: import the contents of a PDF file as plain text
  • Filtered wiki: import from Wikipedia (Wiki), SuperMemopedia (SMP), SuperMemo Guru (Guru) or SuperMemo Help (Help) by using wiki filters
  • Picture: if you import documents recognized as pictures, they will be imported to image components in your default image template of the import concept. Images displayed in Bing or Google will also be qualified as pictures
  • YouTube: if the selected YouTube video is recognized by its YouTube URL, it will be imported as a YouTube script to be watched in the incremental video process. Live videos can also be imported and watched incrementally (during or after the broadcast)

Some experience is needed to optimally choose between HTML (for neatly formatted pages), Parsed HTML (for pages packed with advertising) and plain text (for clean and compact imports). If you are not sure which one to choose, import as many variants as you have available and delete all but the one that you are most happy with. After a month or two, you will instantly recognize which pages import best in which mode. There will still be pages that do not import satisfactorily. For those, you can reserve Link or (Plain) Text imports. At first review, you can resort to old copy-and-paste to replace the import or its poorly formatted portions.

Importing parsed HTML is equivalent to using Zander Parser, which can also be used by other versions of SuperMemo.

After importing a link, it can be opened in the default browser by right-clicking over the Reference link button in the navigation bar and choosing Open hyperlink, or with Open in new window over the link. The link can also be opened in Internet Explorer with Shift+click.

Unlike SuperMemo 18, SuperMemo 19 takes and compares two sources of data: (1) plain text delivered by the web browser, and (2) original HTML from the source URL. Those two are often not well synchronized, which poses a difficulty in re-assembling clean text. SuperMemo needs to resort to a set of heuristics to resolve differences. The main disadvantage is that it will occasionally import both versions of text for the user to choose from. The main advantage is cleaner imports that respect selections, skip some advertising, use plain text in places where it is easier to use than HTML, etc.

It is occasionally possible to import plain text even if texts are behind a paywall.

Importing tabs

[edit | edit source]

To import browser tabs selected in the import list, click the big button Import.

Pages will be imported to the currently used collection. The import will go to the concept group selected at the top of the window. Import node will be titled as the text chosen in Title (if you do not define this text, SuperMemo will generate it using imported page titles). Each page will be imported in its own chosen import mode (e.g. parsed, plain text, link, etc.). Pages marked with the Close icon will close their corresponding tabs in the web browser (this may not work on vertical tabs). Depending on the context, if there are any texts selected in the browser, only the selection will be imported.

Selecting texts for import

[edit | edit source]

When you select a portion of the text in the web browser, SuperMemo will attempt to import only the selection. It may struggle with hidden texts as they may not show up as part of the selection. It then uses a heuristic to choose between the selection and the formatted body of the text. When in doubt, both will be imported so that you can edit out the part that is not needed.

Import window description

[edit | edit source]

The Web import dialog box consists of the following components:

Controls

[edit | edit source]
  • Title - if you choose Location : New node (below), the contents of this field and the date/time will form the title of the node that will parent the imported resources. You can click the Autogenerate title for selected tabs toolbar button to automatically fill it out based on the titles of the sources that are selected in the import list
  • Concept group - concept group into which the selected resources will be imported
  • Import - import selected browser tabs into the currently opened collection
  • Help - display this help page
  • list of tabs that are identified as valid import documents (see the list of supported web browsers and visibility limitations such as hidden toolbars, fullscreen view, vertical tabs, etc.)
    • No - position of the import tab in the list (i.e. number of the row)
    • Sel - checkmark indicating a document that should be imported (when Import is clicked)
    • Dup - duplicate icon (documents that have already been imported are marked with the red icon, documents that are tab duplicates are denoted with the blue one)
    • Title - title of the resource
    • Address - URL of the resource (e.g. page, picture, YouTube video, etc.)
    • Text - text preview (if collected); to attempt to retrieve the preview, click this field
    • Mode - import mode (e.g. HTML, plain text, parsed, etc.)
    • Close - denotes whether the corresponding web browser tab should be closed after the import. Warnings!: This may not work on vertical tabs. Pages that are not selected for import get closed as well.
    • Browser - web browser detected (not all browsers can be detected, not all browsers detected provide the full range of options described in this document). This will be used in the #Source field of the reference area that is automatically created for imported articles.
  • Text preview - the plain text contents of the selected importable. Click on its title or address (in the import list) to get it.
  • Location - determines the location in the Contents window where selected resources will be imported
    • New node - the selected rows will be imported as children of a newly added element (called import node). Note that you can specify the title for this element using the Title field at the top of the dialog box (above)
    • Group hook - the selected rows will be imported as children of the hook of the currently selected concept group. Note that you can change the concept group to which the selected rows will be imported with the Concept group text field (above)
    • Child - the selected rows will be imported as children of the currently displayed element
  • Priority - determines the priority of the imported resources. Elements will have their priorities dispersed randomly in the following range:
    • Minimum priority - specifies the priority at the low end of the range (as percentage). Note that you can set the minimum priority as the position in the priority queue by adding p (e.g. 33p means minimum priority at position 33 in the priority queue)
    • Maximum priority - specifies the priority at the high end of the range (as percentage). Note that you can set the maximum priority as the position in the priority queue by adding p
  • Create concept - if you choose Location : New node and select this checkbox, SuperMemo will import the selected importables as children of a newly added import node that will be converted into a concept (the Title field will serve as its name)
  • Remember settings - reuse the current settings (e.g. priorities) in future imports
  • Auto-Play (YouTube) - help with turning off YouTube autoplay in Chrome (to facilitate massive imports). If Edge is your browser of choice, copy and paste edge://settings/privacy/sitePermissions/allPermissions/mediaAutoplay into your browser's address bar, and choose Block for Control if audio and video play automatically on sites

PDF

[edit | edit source]

SuperMemo 19 does not support formatted PDF imports nor PDF editing. For this, you can import PDFs using methods used in SuperMemo 18. However, you can alternatively import PDFs as plain text with the original file retained as a link. This way, while reading incrementally, you can complete poorly formatted or missing parts of the document (e.g. mathematical formulas, pictures, etc.).

The main advantages of PDF text import are:

  • importing directly from the web browser in the same way as HTML
  • instant availability of the full text for search in incremental learning
  • small size of the document for cases where you import many documents as archives (rather than the material for incremental reading)

To import PDF as text, see if the file was recognized as PDF (in the Mode column in the import list). If not, right-click of the row, and choose Text (Ctrl+T) to indicate the file should be imported as plain text (marked as Text in the Mode column)

Problems

[edit | edit source]

If a web browser tab cannot be closed, its title and URL will remain displayed in the import list (this mostly happens if the tab is closed between collecting and importing tags).

Imports above 100 tabs are annoyingly slow. In some cases, it might block the web browser. If Edge asks "Restart MS Edge?", restarting won't likely affect the import apart from accelerating it (due to releasing some of the resources used).

Close selected or Close all can be carried out by:

  1. selecting importables,
  2. marking Close,
  3. marking Ignore, and
  4. clicking Import

If you make tabs invisible (e.g. by turning the full-screen mode on with F11), SuperMemo will ignore them. If a YouTube player is in full screen, SuperMemo will also ignore it. The same happens for many other apps.

MS Edge vertical tabs can be detected, some extra tabs may be included (e.g. YouTube panel tabs), but they aren't closed on import. Press Ctrl+Shift+, in Edge to turn the vertical tabs off before collecting them for import in SuperMemo.

简介

[编辑|编辑源代码]

网页导入允许从网络大量导入知识。如果您的好奇心让您打开一百个选项卡，您可以在几秒钟内将浏览器中的混乱内容转换为组织良好的学习材料。

如果您将网络导入与增量阅读，您将永远不需要限制您在网络上的探索。您永远不会遇到影响大多数活跃在社交媒体上的人的生产力下降的问题。

您可以将学习的乐趣分为（1）探索和（2）消费。您可以在可用时间内随意探索，然后将所有结果传输到超级备忘录。在后来的时间里，与增量阅读，您可以处理所有导入的知识，以最大限度地提高学习效益。

导入方式

[编辑|编辑源代码]

导入方式：

  • 超文本标记语言：在此默认模式下，原始网页将从网络上拉下来，过滤并导入到SuperMemo（对于需要密码的页面，纯文本需要使用）
  • 选择：从文本中导入所选部分的文本网络浏览器。如果可以检索格式化的选择，则选择的格式将与浏览器中的一样。在某些情况下，由于动态插入的内容，SuperMemo 可能需要在格式化版本和纯文本之间进行选择。当不确定时，SuperMemo 将导入所选内容的两个版本
  • 纯文本: 选择文字导入纯文本的模式；该模式也可以在紧急情况下使用，例如当您失去互联网连接时
  • 纯文本选择：选择文本中的一部分网络浏览器，并选择文字导入中的模式用于导入纯文本选择（例如格式丰富的长篇文章的部分）。如果无法检索选择，将导入整个文本（您将能够在文本预览中看到事实）。当文本容器不支持所有文本选择功能时会发生这种情况
  • 链接：仅导入文档的链接（用于在需要时增量导入）
  • 已解析的文档（解析)：导入不含非必要部分（页眉、页脚、广告等）的 HTML 文档。文本中的选择将被忽略
  • 纯文本 PDF：将 PDF 文件的内容导入为纯文本
  • 过滤维基：从维基百科导入（维基百科），超级记忆百科（表面活性剂), SuperMemo 大师 (大师) 或 SuperMemo 帮助 (帮助）通过使用 wiki 过滤器
  • 图片：如果导入识别为图片的文档，它们将被导入到导入概念的默认图像模板中的图像组件中。在 Bing 或 Google 中显示的图像也将被视为图片
  • YouTube：如果所选 YouTube 视频可以通过其 YouTube URL 识别，则会将其导入为 YouTube 脚本，以便在增量视频过程。直播视频也可以导入并增量观看（在广播期间或之后）

需要一些经验才能在 HTML（用于格式整齐的页面）、解析的 HTML（用于包含广告的页面）和纯文本（用于干净且紧凑的导入）之间进行最佳选择。如果您不确定选择哪一种，请导入尽可能多的可用变体，然后删除除您最满意的变体之外的所有变体。一两个月后，您将立即识别哪些页面在哪种模式下导入效果最好。仍然会有一些页面导入效果不理想。对于这些，您可以预订链接或（纯）文本进口。第一次检查时，您可以使用旧的复制粘贴来替换导入或其格式不良的部分。

导入已解析的 HTML 相当于使用桑德尔解析器，其他版本的 SuperMemo 也可以使用。

导入链接后，可以通过右键单击该链接在默认浏览器中打开该链接参考链接按钮在导航栏并选择打开超链接，或与在新窗口中打开通过链接。该链接也可以在 Internet Explorer 中打开Shift+单击。

与 SuperMemo 18 不同，SuperMemo 19 获取并比较两个数据源：(1) Web 浏览器提供的纯文本，以及 (2) 来自源 URL 的原始 HTML。这两者通常不能很好地同步，这给重新组装干净的文本带来了困难。 SuperMemo 需要诉诸一系列启发式方法来解决差异。主要缺点是它偶尔会导入两个版本的文本供用户选择。主要优点是更干净的导入，尊重选择，跳过一些广告，在比 HTML 更容易使用的地方使用纯文本等。

即使文本位于付费墙后面，有时也可以导入纯文本。

导入标签

[编辑|编辑源代码]

要导入在导入列表中选择的浏览器选项卡，请单击大按钮进口。

页面将被导入到当前使用的收藏。导入将转到概念组在窗口顶部选择。进口节点将标题为中选择的文本标题（如果您没有定义此文本，SuperMemo 将使用导入的页面标题生成它）。每个页面都将以自己选择的导入模式导入（例如解析、纯文本、链接等）。标有的页面关闭图标将关闭网络浏览器中相应的选项卡（这可能不适用于垂直选项卡）。根据上下文，如果在浏览器中选择了任何文本，则仅导入选择的内容。

选择要导入的文本

[编辑|编辑源代码]

当您选择文本中的一部分时网络浏览器，SuperMemo 将尝试仅导入所选内容。它可能会遇到隐藏文本的问题，因为它们可能不会显示为选择的一部分。然后，它使用启发式方法在文本的选择内容和格式化正文之间进行选择。如有疑问，两者都将被导入，以便您可以编辑掉不需要的部分。

PDF

[编辑|编辑源代码]

SuperMemo 19 不支持格式化 PDF 导入，也不支持 PDF 编辑。为此，您可以使用导入 PDFSuperMemo 18 中使用的方法。但是，您也可以将 PDF 作为纯文本导入，并保留原始文件作为链接。这样，在增量阅读的同时，您可以完成文档中格式不良或缺失的部分（例如数学公式、图片等）。

PDF文本导入的主要优点是：

  • 直接从网络浏览器与 HTML 相同的方式
  • 即时提供全文供搜索渐进学习
  • 如果您导入许多文档作为存档（而不是材料），则文档尺寸较小增量阅读）

要将 PDF 导入为文本，请查看该文件是否被识别为 PDF（在模式导入列表中的列）。如果没有，请右键单击该行，然后选择文字（Ctrl+T）指示文件应以纯文本形式导入（标记为文字在模式栏）

问题

[编辑|编辑源代码]

如果无法关闭 Web 浏览器选项卡，其标题和 URL 将保留显示在导入列表中（如果在收集和导入标签之间关闭选项卡，通常会发生这种情况）。

导入超过 100 个选项卡的速度慢得令人烦恼。在某些情况下，它可能会阻止网络浏览器。如果边缘问“重新启动 MS Edge？”，除了加速导入之外（由于释放了一些使用的资源），重新启动不太可能影响导入。

关闭所选或全部关闭可以通过以下方式进行：

  1. 选择进口物品，
  2. 标记关闭,
  3. 标记忽略, 和
  4. 点击进口

如果您使选项卡不可见（例如，通过使用以下命令打开全屏模式）F11），SuperMemo 会忽略它们。如果一个YouTube播放器处于全屏状态，SuperMemo 也会忽略它。许多其他应用程序也会发生同样的情况。

微软边缘垂直标签可以检测到，可能会包含一些额外的选项卡（例如 YouTube 面板选项卡），但它们在导入时不会关闭。新闻Ctrl+Shift+,在 Edge 中关闭垂直选项卡，然后再收集它们以导入 SuperMemo。
