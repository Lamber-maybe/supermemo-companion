Unlocking commercial SuperMemo

[edit | edit source]

If SuperMemo asks you for a password or there is a red registration bar below the navigation bar, you are using a locked copy of SuperMemo:

This red registration bar indicates that SuperMemo is locked:

Click here to register SuperMemo (credit used: 23%)

To unlock SuperMemo you will need an unlock password.

To receive the unlock password, click on the red bar, fill in your Order ID, and send your Machine identification code to SuperMemo World (automatically or manually via e-mail at address register2026(@)supermemo.org).

To receive the password, you need to send your registration data via e-mail.

If you do not remember your order number, you can skip it as long as you use the same e-mail address as when ordering. You can also skip your Order ID if you are using a trial copy of SuperMemo.

Figure: Registering SuperMemo dialog, available upon clicking the red registration bar at the top of the element window, assists you in submitting your registration data in order to receive the unlock password

You will receive the unlock password within a business day. Until then you can still fully explore SuperMemo without the password. You are unlikely to run out of credit in less than a week.

To unlock SuperMemo, input the unlock password received in response to your order with File : Installation : Input password (Ctrl+I). Use can also click the button Password in the registration dialog used for sending your password request.

You are unlikely to ever need to input your password twice unless you move to a new computer. In such a case, register your new computer using the same procedure as above.

Receiving assistance

[edit | edit source]

If you need assistance, or need your password resent, or you want to register on a new computer, get in touch via e-mail.

Important! If you do not receive the unlock password within two business days, please get in touch via e-mail. The most likely reason is that your mail was blocked by a spam filter, it was lost in delivery, or you have misspelled your e-mail address when purchasing the program. In case you cannot get in touch via e-mail, please send us an alternative e-mail address, or simply leave your message at SuperMemoPedia.

If you do not like the whole concept of commercial applications, passwords, etc., you can use SuperMemo Freeware.

Mail options at registering

[edit | edit source]

If you press Send in the registration dialog, SuperMemo will yet ask you to choose your mail client.

If you are not sure what to choose, pick the default and ignore mail formatting (if unreadable).

You have the following options:

  • Send mail with spaces and line breaks (e.g. for Thunderbird). This option will encode any characters that are not allowed in a web address, such as spaces or line breaks, into a code that can be transmitted over the internet. For example, a space will be replaced by %20, and a line break will be replaced by %0D%0A. This option may work well with email clients that can decode these codes and display them correctly, such as Thunderbird. However, some email clients may not recognize these codes and display them as they are, making the message unreadable, such as Gmail.
  • Use the default MAPI mailer on your system (e.g. Outlook). This option will use the MAPI protocol to communicate with the email client that is set as the default on your system, such as Outlook. MAPI stands for Messaging Application Programming Interface, and it is a standard way of sending and receiving emails on Windows. This option may work well with email clients that support MAPI. However, some email clients may not support MAPI or may have some compatibility issues.
  • Format mail as a web page using HTML. This option will use HTML tags to format the message as a web page, such as using <br> or <p> for line breaks. HTML is a standard way of creating and displaying web pages. However, some email clients may not support HTML tags or may show them as plain text, such as Windows Live Mail.
  • Generate a file for sending later. This option will create a file that contains the registration message. You can send it later. This option may give you more flexibility and control over how you send your message. It is the same as clicking the button File in the registration dialog. That file is stored in the collections \reports folder as register.txt (e.g. c:\SuperMemo\systems\Physics\reports\register.txt)

Unlocking a demo of SuperMemo

[edit | edit source]

When you use a demo version of SuperMemo, you are given a credit of 100 points. Once this credit is used up, by using SuperMemo, you will be asked to input the unlock password. The credit available is displayed in the parentheses on the red registration bar (see above).

You can use the following procedure to get help in unlocking your demo version of SuperMemo:

  1. Click the red registration bar to open the registration dialog box
  2. Fill out the fields Name, Country and Contact
  3. Click Send

Those steps will result in sending an e-mail with your data. In response you will receive further instructions (e.g. how to get the unlock password, how to get SuperMemo in return for writing learning material, how to buy a license, etc.).

Spread SuperMemo among your friends

[edit | edit source]

The advantage of using the above complex lock/registration mechanism is that you can spread SuperMemo among your friends.

Important! You can freely pass SuperMemo to your friends. It will automatically get locked on unregistered computers.

If you want to redistribute your own collections, you can use File : Installation : Lock before putting your collections on the web or passing them to your friends. If you have questions about redistributing your own collections with SuperMemo, please get in touch.

We apologize for the inconvenience related to the registration process!

The goal of SuperMemo World is to popularize SuperMemo. We are happy to welcome every new member in the SuperMemo family. We would love to provide customers with a free working version of SuperMemo. At the same time, we could not forget that the company has to earn for its operations. On one hand, we cannot make all SuperMemos free, on the other, we cannot reward "clever" users with an easy way to circumvent the license fee, as we do not want to contribute to a world where it pays to live by hook or by crook.

Our experience shows that the shareware approach does not work for SuperMemo. In the years 1991-2016, we have gained thousands of users of freeware versions of SuperMemo 2, SuperMemo 5, SuperMemo 6, SuperMemo 7, SuperMemo 9, SuperMemo 12, SuperMemo 15, and SuperMemo 16. At the same time, we received just a handful of shareware contributions while the number of support requests was a thousand times greater!

Remember! If you cannot afford SuperMemo, you can always use a freeware version or get the latest version in return for your participation in beta-testing or similar programs.

注册时的邮件选项

[编辑|编辑源代码]

如果你按发送在注册对话框中，SuperMemo 仍会要求您选择邮件客户端。

如果您不确定选择什么，请选择默认值并忽略邮件格式（如果不可读）。

您有以下选择：

  • 发送带有空格和换行符的邮件（例如雷鸟）。此选项会将网址中不允许的任何字符（例如空格或换行符）编码为可以通过互联网传输的代码。例如，空格将替换为 %20，换行符将替换为 %0D%0A。此选项可能适用于可以解码这些代码并正确显示它们的电子邮件客户端，例如 Thunderbird。但是，某些电子邮件客户端可能无法识别这些代码并按原样显示它们，从而导致邮件无法读取，例如 Gmail。
  • 使用系统上的默认 MAPI 邮件程序（例如 Outlook）。此选项将使用 MAPI 协议与系统上默认设置的电子邮件客户端（例如 Outlook）进行通信。 MAPI 代表消息应用程序编程接口，它是在 Windows 上发送和接收电子邮件的标准方法。此选项可能适用于支持 MAPI 的电子邮件客户端。但是，某些电子邮件客户端可能不支持 MAPI 或者可能存在一些兼容性问题。
  • 使用 HTML 将邮件格式化为网页。此选项将使用 HTML 标记将消息格式化为网页，例如使用 <br> 或 <p> 进行换行。 HTML 是创建和显示网页的标准方式。但是，某些电子邮件客户端可能不支持 HTML 标签或可能将其显示为纯文本，例如 Windows Live Mail。
  • 生成文件以便稍后发送。此选项将创建一个包含注册消息的文件。您可以稍后发送。此选项可以让您更加灵活地控制发送消息的方式。和点击按钮是一样的文件在注册对话框中。该文件存储在收藏品 \报告文件夹为注册.txt（例如c:\SuperMemo\systems\Physics\reports\register.txt）

解锁 SuperMemo 演示

[编辑|编辑源代码]

当您使用 SuperMemo 的演示版时，您将获得 100 点积分。一旦此积分用完，使用 SuperMemo 时，系统会要求您输入解锁密码。可用积分显示在红色注册栏的括号中（见上文）。

您可以使用以下过程来获取解锁 SuperMemo 演示版的帮助：

  1. 点击红色注册栏，打开注册对话框
  2. 填写字段名称,国家和联系方式
  3. 点击发送

这些步骤将导致发送一封包含您的数据的电子邮件。作为回应，您将收到进一步的说明（例如，如何获取解锁密码、如何获得 SuperMemo 以换取编写学习材料、如何购买许可证等）。
