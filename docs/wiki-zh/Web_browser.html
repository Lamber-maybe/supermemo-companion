Web browser is an application for exploring the web. SuperMemo will help you collect the results of your explorations in SuperMemo collections. When the number of opened tabs becomes unmanageable, use Web import. If you select content in the browser, Web import may ignore the rest of the texts in the imported content.

At the moment of release (SuperMemo 19 in September 2023), SuperMemo supported Microsoft Edge, retained support for Internet Explorer, and added basic imports from Google Chrome. Further Chrome functionality will be added in coming months. If you use another browser that you want supported in SuperMemo, please get in touch via e-mail.

在发布时（2023 年 9 月的 SuperMemo 19），SuperMemo 支持微软边缘,保留支持对于 Internet Explorer，并添加了基本导入谷歌浏览器。更多 Chrome 功能将在未来几个月内添加。如果您使用希望 SuperMemo 支持的其他浏览器，请通过以下方式联系电子邮件。
