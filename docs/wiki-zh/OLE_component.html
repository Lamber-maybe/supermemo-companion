Overview

[edit | edit source]

If you would like to display files created in MS Word, Excel, Equation, PowerPoint, Corel, Mind Manager, etc. you can use the OLE component. The OLE component can display files created with any application that supports OLE 2.0. If this application supports in-place activation, you will also be able to edit these files without quitting SuperMemo. For example, if you insert an MS Word document into an OLE component and switch to editing mode, SuperMemo menus and toolbars will be replaced by Word menus and toolbars.

As all other components, the OLE component is available from the Compose toolbar. To be able to edit files inserted in the OLE component you will need an appropriate server application (e.g. MS Word, Corel, Mind Manager, etc.). However, you do not need any software to display these files in the presentation mode.

To add an OLE component, click OLE button on the Compose toolbar. You might resize and reposition it within the element.

The OLE component will show as an empty rectangle. You may need to turn off Scaled property on the element menu or in your template due to sizing problems with various OLE objects.

If you want to import a ready file, right-click the OLE component, and choose Insert OLE object, click Create from File in the Insert Object dialog box and choose the file by typing its name or clicking Browse.

If you want to create a new file and edit its contents, right-click the OLE component, and choose Insert OLE object, and pick from the list of OLE servers available in your Windows installation. When you click OK, the appropriate server will be called and you will be able to edit your file (e.g. Microsoft equation, Excel chart, PowerPoint presentation, etc.). When you press Esc, SuperMemo will display the file in the presentation mode (i.e. the way it will look during repetitions or on the computer of the users of your collection).

Please note that the pop-up menu in the OLE component is mode sensitive, i.e. it will display the OLE server pop-up menu in the editing mode and standard SuperMemo options in the presentation mode. In the presentation mode you will be able to use options such as Display At, Answer, etc. In the editing mode you will be able to use options depending on your OLE server (e.g. if your server is MS Excel, you will be able to use Cut, Copy, Format cells, etc.).

The simplest way to switch between presentation and editing modes is to choose one of the following:

  • Alt+click over the element display area
  • press Ctrl+T until you reach the OLE component
  • click the Editing mode button (blue-notebook) on the Edit toolbar (Ctrl+E)

Question

I'm having troubles with large mindmaps: I can't see the entire one. What's the solution?

Answer

If you are using mind-maps with OLE, their behavior will depend on the mind-map OLE server. If you are using mind-maps with HTML or with the image component, you can view the file with F9.

It is important to know that in active recall, you only need a small portion of your mind-map to efficiently execute your repetitions. Ideally, you should convert your mind-map to a graphic file and paste only small portions of this file as separate topics (alternatively, you can use the image component's zoom&trim mode). Then, you can use questions referring to individual portions of the map or use occlusion tests to cover parts of the map. Asking questions about the mind-map is by far simpler and often more effective. Occlusion tests are harder to build but easier to recall. Consequently, you may want to use occlusion tests on large volume material where speed of learning is more important than precise map recall.

In any case, large mindmap preview in SuperMemo should not be necessary

Question

I wanted to import PDF files to SuperMemo for reading and copying certain items for further analysis. I initially tried to use Insert OLE object but I found OLE mechanisms unstable

Answer

Another approach is to use the web browser (HTML) component. Then use Insert HTML link : HTML file (on the HTML component pop-up menu) and link to the PDF file. Using this option the PDF file is not immediately visible as only files of type *.txt, *.htm, *.rtf are listed. To visualize the PDF files on your drive type in *.pdf in the file name field and they become visible in the File window. Naturally, you need to have Acrobat Reader installed to view the file in SuperMemo. You can, of course, copy the text with the tools provided for the PDF viewer and use it for further SuperMemo purposes. The same method works for Excel files, Word files, etc.

Question

When I create an MS Equation formula in the OLE component, the equation is unnaturally large and distorted. How can I go around this bug?

Answer

Rendering of the OLE object is done by the server application, MS Equation in this case. You have to fix the problem in MS Equation by adjusting display attributes (e.g. reducing the size of fonts in this case)

概述

[编辑|编辑源代码]

如果您想显示在 MS Word、Excel、Equation、PowerPoint、Corel、心灵经理等等，你可以使用OLE组件。 OLE 组件可以显示使用任何支持 OLE 2.0 的应用程序创建的文件。如果此应用程序支持就地激活，您还可以在不退出 SuperMemo 的情况下编辑这些文件。例如，如果将 MS Word 文档插入 OLE组件并切换到编辑模式、SuperMemo 菜单和工具栏将被 Word 菜单和工具栏取代。

与所有其他组件，OLE 组件可从撰写工具栏。能够编辑 OLE 中插入的文件组件您将需要适当的服务器应用程序（例如 MS Word、Corel、Mind Manager 等）。但是，您不需要任何软件即可在演示模式。

添加 OLE组件，点击奥莱按钮上的撰写工具栏。您可以调整其大小并重新定位它元素。

欧莱雅组件将显示为空矩形。您可能需要关闭缩放财产在元素菜单或者在你的模板由于各种 OLE 对象的大小问题。

如果要导入准备好的文件，请右键单击 OLE组件，并选择插入 OLE 对象，点击从文件创建在插入对象对话框，然后通过键入文件名或单击来选择文件浏览。

如果要创建新文件并编辑其内容，请右键单击 OLE组件，并选择插入 OLE 对象，然后从 Windows 安装中可用的 OLE 服务器列表中进行选择。当您点击好的，将调用相应的服务器，您将能够编辑您的文件（例如 Microsoft 方程、Excel 图表、PowerPoint 演示文稿等）。当您按下ESC，SuperMemo 将在演示模式（即它在重复过程中或在您的用户的计算机上的外观收藏）。

请注意 OLE 中的弹出菜单组件是模式敏感的，即它将在编辑模式和标准 SuperMemo 选项演示模式。在演示模式下，您将能够使用诸如显示于,回答等。在编辑模式您将能够根据您的 OLE 服务器使用选项（例如，如果您的服务器是 MS Excel，您将能够使用切,复制,设置单元格格式等）。

在演示和演示之间切换的最简单方法编辑模式是选择以下一项：

  • Alt+单击在元素显示区域上方
  • 按Ctrl+T直到到达 OLE组件
  • 单击编辑模式按钮（蓝色笔记本）编辑工具栏（Ctrl+E）

问题

我在处理大型思维导图时遇到了麻烦：我看不到整个思维导图。解决办法是什么？

回答

如果您将思维导图与 OLE 结合使用，它们的行为将取决于思维导图 OLE 服务器。如果您使用带有 HTML 或图像的思维导图组件，您可以使用以下命令查看文件F9。

重要的是要知道，在主动回忆，你只需要思维导图的一小部分就可以有效地执行你的重复。理想情况下，您应该将思维导图转换为图形文件，并仅将该文件的一小部分粘贴为单独的主题（或者，您可以使用图像组件的缩放和修剪模式）。然后，您可以使用涉及地图各个部分的问题或使用遮挡测试以覆盖地图的部分区域。询问有关思维导图的问题要简单得多，而且通常更有效。闭塞测试更难构建但更容易回忆。因此，您可能想要使用遮挡测试在大量材料上，学习速度比精确的地图回忆更重要。

无论如何，SuperMemo 中的大型思维导图预览不是必需的
