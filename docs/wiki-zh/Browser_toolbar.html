Browser

[edit | edit source]

Browser toolbar appears at the top of the browser window.

Buttons

[edit | edit source]

Here are the functions of individual toolbar buttons from left to right:

  1. Main menu (Shift+F10) - open the browser menu otherwise available with a right-click over the browser window
  2. Subset menu - open a subset processing menu with operations on browser elements (same as Process browser> on the browser menu available with a right-click)
  3. Align - align the browser window with the element window for convenient review of elements in the browser. Note that the Synchronize button (below) must be down for displaying elements in the element window as the selection changes in the browser
  4. Review all (Ctrl+Shift+L) - review all material in the browser, e.g. before an exam (including elements that are not outstanding)
  5. Review topics - review all topics in the browser (including non-outstanding topics)
  6. Go neural (Ctrl+F2) - start a neural review using the browser contents or add the contents to the current review (if any)
  7. Checked - open a child browser with all checked elements (checkmarks can be seen in the leftmost column of the browser).
  8. Unchecked - open a child browser with all unchecked elements
  9. Memorized - open a new browser with all memorized elements included in the present browser
  10. Items - open a new browser with all items included in the present browser
  11. Topics - open a new browser with all topics included in the present browser
  12. Sources - open sources of all elements in the browser. Sources are top-most topics from which sub-topics, extracts, sentences, cloze deletions and/or Q&A items have been generated in incremental reading
  13. Sort by tree - sort the browser by the position of its elements in the contents tree (root first, first child of the root second, etc.)
  14. Spread priorities - set the priority of all elements in the browser (increase, decrease, or spread within a selected range)
  15. Advance topics - shorten the interval of topics listed in the browser. If you specify the maximum interval as 100, all topics with longer interval will receive intervals randomly dispersed within the 1..100 range. Topics with intervals shorter than 100 will remain untouched. Advance topics executes mid-interval repetition on the affected topics (unlike Advance items). See: Advancing elements
  16. Advance items - shorten the intervals of items listed in the browser. If you specify the maximum interval as 1000, all items with interval longer than 1000 will receive intervals randomly shortened so that to schedule a repetition between today or the day when the interval reaches 1000 days. All items repeated before 1000 days ago, will be scheduled for repetition today. Unlike Advance topics, Advance items shortens intervals without executing a mid-interval repetition (which would be illegal from the learning process standpoint as no repetition can be considered executed without you actually reviewing it). Important! Abusing this option can lead to serious side effects in learning such as inadvertently sending important items to very long intervals by lumped repetition of similar items in a very short period of time. See: Advancing elements
  17. Postpone - postpone all outstanding elements included in the browser
  18. Insert picture - add a selected picture to all elements in the browser
  19. Visual browser - switch between table and thumbs modes (only in Windows Vista or later). In the thumbs mode, the browser displays element snapshots as thumbs (i.e. miniature images). The thumbs mode is useful when browsing through picture-rich collections (e.g. photo albums) (see: picture)
  20. Apply template - apply a selected template to all elements in the browser
  21. Randomize (Ctrl+Shift+F11) - randomly mix up the contents of the browser. You can, for example, randomize the browser before scheduling its content for repetition with Schedule repetitions (below)
  22. Schedule repetitions - save the contents of the browser in today's repetition schedule. You will most likely want to use this button for optimizing the sequence of outstanding repetitions. You may, for example, begin the sequence with short-interval items, or with difficult items, or with most severely delayed items, or with long-interval topics, etc.
  23. Find (Alt+F and F3) - search for texts in the title column starting from the currently selected element
  24. View (Enter or double-click) - display the currently selected element in the element window. Note that you will not need this button if the Synchronize button (below) is down, in which case the element window will always display the element currently selected in the browser
  25. Save subset - save the contents of the browser into a subset file
  26. Synchronize - makes sure that each time you change the selection in the browser, the selected element is displayed in the element window. When this button is up, changing the selection in the browser has no effect on the content of the element window

浏览器

[编辑|编辑源代码]

浏览器工具栏出现在顶部浏览器窗口。

按钮

[编辑|编辑源代码]

以下是从左到右各个工具栏按钮的功能：

  1. 主菜单（转变+F10) - 打开浏览器菜单否则可以通过右键单击浏览器窗户
  2. 子菜单- 打开一个子集处理菜单在浏览器上进行操作元素（与流程浏览器>于浏览器菜单右键单击即可使用）
  3. 对齐- 对齐浏览器窗口与元素窗口为了方便查阅元素在浏览器。请注意，同步按钮（下面) 必须按下才能显示元素在元素窗口随着选择的变化浏览器
  4. 查看全部（控制键+转变+L) - 审查所有材料浏览器，例如考试前（包括元素那些不是杰出的）
  5. 回顾主题- 审查所有主题在浏览器（包括非突出主题）
  6. 神经化（控制键+F2) - 使用以下命令开始神经审查浏览器内容或将内容添加到当前评论（如果有）
  7. 已检查- 打开一个孩子浏览器全部检查完毕元素（复选标记可以在最左边的列中看到浏览器）。
  8. 未选中- 打开一个孩子浏览器一切未经检查元素
  9. 已熟记- 打开一个新的浏览器与所有记忆的元素包含在现在的浏览器
  10. 项目- 打开一个新的浏览器与所有项目包含在现在的浏览器
  11. 主题- 打开一个新的浏览器与所有主题包含在现在的浏览器
  12. 来源- 全部开源元素在浏览器。来源是最上面的主题从哪些子主题，提取物, 句子,完形填空删除和/或问答项目已生成于增量阅读
  13. 按树排序- 排序浏览器由其地位元素在内容树（根第一，第一孩子根秒等）
  14. 分散优先事项- 设置优先级全部的元素在浏览器（在选定范围内增加、减少或分散）
  15. 高级主题- 缩短间隔的主题列出在浏览器。如果您指定最大间隔为 100 时，全部主题间隔较长的将收到随机分布在 1..100 范围内的间隔。主题与间隔小于 100 的将保持不变。高级主题执行中间间隔重复对受影响的主题（不像预售项目）。参见：先进元素
  16. 预售项目- 缩短间隔的项目列出在浏览器。如果您指定最大间隔为 1000 时，全部项目间隔大于 1000 的间隔将随机缩短，以便安排在今天或间隔达到 1000 天的那天之间重复。全部项目1000 天前重复过的，今天将安排重复。不像高级主题,预售项目缩短间隔不执行中间间隔重复（从学习过程的角度来看，这是非法的，因为在没有实际复习的情况下，不能认为重复已执行）。重要！滥用此选项可能会导致学习中的严重副作用，例如无意中发送重要信息项目很长间隔通过在很短的时间内集中重复类似的项目。参见：先进元素
  17. 推迟- 推迟所有突出元素包含在浏览器
  18. 插入图片- 将选定的图片添加到所有图片中元素在浏览器
  19. 可视化浏览器- 在桌面模式和拇指模式之间切换（仅在 Windows Vista 或更高版本中）。在拇指模式下，浏览器将元素快照显示为拇指（即微型图像）。浏览丰富图片时，拇指模式非常有用收藏品（例如相册）（参见：图片）
  20. 应用模板- 应用选定的模板致所有人元素在浏览器
  21. 随机化（控制键+转变+F11) - 随机混合内容浏览器。例如，您可以随机化浏览器在安排其内容重复之前安排重复次数（下）
  22. 安排重复次数- 保存内容浏览器在今天的重复计划中。您很可能希望使用此按钮来优化未完成重复的顺序。例如，您可以用简短的序列开始间隔 项目，或有困难的项目，或有最严重延误的项目，或有长间隔主题等
  23. 查找（替代+F和F3) - 从当前所选内容开始在标题列中搜索文本元素
  24. 查看（输入或双击) - 显示当前选择的元素在元素窗口。请注意，如果同步按钮（下方）按下，在这种情况下元素窗口将始终显示元素当前选择的浏览器
  25. 保存子集- 保存内容浏览器到子集文件中
  26. 同步- 确保每次更改选择时浏览器，所选的元素显示在元素窗口。当此按钮弹起时，更改中的选择浏览器对内容没有影响元素窗口
