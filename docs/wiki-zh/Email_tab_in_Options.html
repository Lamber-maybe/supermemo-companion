E-mail tab in Toolkit : Options provides options for sending email from SuperMemo.

  • Default mail protocol
    • Default mail client - SuperMemo will send simple email using your default mail program
    • Secure SMTP (with formatting and attachments) - you may use SuperMemo as your mailing program using the secure SMTP protocol. Despite the name, the method is not secure as SuperMemo is not a safe guardian of your private data (e.g. mail password). Use this method only if the other two cannot be used
    • Simple MAPI (with attachments) - as in previous SuperMemos, if your mailing program supports MAPI, use it to maximize the range of available options, esp. if you forward your elements or objects to friends
  • SMTP data - data needed to use Secure SMTP
    • Username - the username used to authenticate with the SMPT mail server
    • Password - the password used to authenticate with the SMPT mail server. Warning! SuperMemo saves your password using weak encryption. Do not trust SuperMemo with mission-critical data!
    • SMTP host - the name of the SMPT mail server (e.g. smpt.gmail.com)
    • Port - the port used to submit an outgoing email to the SMPT mail server (e.g. 587 for encrypted email transmissions using SMTP Secure (SMTPS))

电子邮件标签进入工具包 :选项提供从 SuperMemo 发送电子邮件的选项。

  • 默认邮件协议
    • 默认邮件客户端- SuperMemo 将使用您的默认邮件程序发送简单的电子邮件
    • 安全 SMTP（带格式和附件）- 您可以使用 SuperMemo 作为您的邮件程序，使用安全的邮件传输协议协议。尽管有这个名字，但该方法并不安全，因为 SuperMemo 不能安全地保护您的私人数据（例如邮件密码）。仅当其他两种方法无法使用时才使用此方法
    • 简单MAPI（带附件）- 与之前的 SuperMemos 一样，如果您的邮件程序支持映射接口，使用它来最大化可用选项的范围，尤其是。如果你转发你的元素或反对朋友
  • SMTP数据- 使用时需要的数据安全 SMTP
    • 用户名- 用于向 SMPT 邮件服务器进行身份验证的用户名
    • 密码- 用于向 SMPT 邮件服务器进行身份验证的密码。警告！SuperMemo 使用弱加密保存您的密码。不要将关键任务数据交给 SuperMemo！
    • SMTP主机- SMPT 邮件服务器的名称（例如smpt.gmail.com）
    • 港口- 用于向 SMPT 邮件服务器提交外发电子邮件的端口（例如第587章用于使用 SMTP Secure (SMTPS) 进行加密的电子邮件传输）
