The View submenu on the main menu is available only on the Middle or higher levels. It makes it possible to open an element browser with various element subsets. The following options are available:

View menu items

[edit | edit source]

Outstanding

[edit | edit source]
  • Outstanding - open the browser with all elements scheduled for today's review. These include today's repetitions as well as repetitions from all recent days on which learning was not completed

Link list

[edit | edit source]
  • Link list - open the browser with the list of concepts and elements linked to the element currently displayed in the element window

Neural queue

[edit | edit source]
  • Neural queue - open the browser with the current neural queue (or build a neural queue for the element currently displayed in the element window)

Priority queue

[edit | edit source]
  • Priority queue - open the browser with the entire priority queue (starting from elements of the highest priority: 0%)

All

[edit | edit source]
  • All - open the browser with all elements stored in the current collection

Memorized

[edit | edit source]
  • Memorized - open the browser with all elements taking part in the learning process (i.e. elements that have been memorized)

Dismissed

[edit | edit source]
  • Dismissed - open the browser with all elements that have been dismissed, i.e. definitely removed from the learning process (e.g. by pressing Ctrl+D in the element window)

Concepts

[edit | edit source]
  • Concepts - open the browser with all elements that are concepts (see: Items, topics, concepts, and tasks)

Topics

[edit | edit source]
  • Topics - open the browser with all elements that are topics (see: Items, topics, concepts, and tasks)

Items

[edit | edit source]
  • Items - open the browser with all elements that are items (see: Items, topics, concepts, and tasks)

Tasks

[edit | edit source]
  • Tasks - open the browser with all elements that are tasks. All tasks are displayed independent of which tasklist they belong to. To view and sort tasks belonging to an individual tasklist, use the tasklist manager (Toolkit : Tasklist (F4)). To view these tasks in a browser, press Shift+Ctrl+B in the tasklist manager

Last browser

[edit | edit source]
  • Last browser - open the browser with the contents of the most recently used browser

Search results

[edit | edit source]
  • Search results - open the browser with elements collected in the last search operation (executed with Ctrl+F or Search : Find elements)

Subset

[edit | edit source]
  • Subset - open the browser with elements stored in a subset file (extension SUB). Subset files can be created in the browser with options such as Subset : Save all or Select : Save selection, etc. Some subsets are created automatically, for example, unpostponed.sub subset stores elements that have been skipped by auto-postpone. For more see: Using subsets

Filter

[edit | edit source]
  • Filter - open the browser with all elements that meet certain criteria such as the number of repetitions, priority, current interval, forgetting index, status (e.g. pending, memorized, etc.) and more

Recent

[edit | edit source]
  • Recent - open the browser with recently used subset of elements
    • Learned - open the browser with elements recently used in subset learning
    • Reviewed - open the browser with elements subject to the current subset review (e.g. with Review all). You can thus see the elements that have already been reviewed and the queue of elements that are still scheduled for further subset review. The elements that have already been reviewed have their Last Rep field showing the last repetition date (e.g. today). They can easily be filtered out (e.g. with Select : Select browser range on the browser menu)
    • To postpone - open the browser with elements on which a recent Postpone procedure has been executed. Auto-postpone does not create this subset as all outstanding elements are subject to auto-postpone. Not all elements in To postpone subset will actually be postponed (e.g. due to not meeting postpone criteria). The subset file is: subsets\postponing.sub
    • Postponed - open the browser with elements whose intervals have recently been extended with Postpone. This subset can be created with manual Postpone or with Auto-postpone. After Postpone this subset will contain postponed elements and will be smaller or equal to To postpone subset. To see which elements have been auto-postponed, use Postponed. The subset file is: subsets\postponed.sub
    • Not postponed - open the browser with elements that have been skipped in the recent postpone operation. The subset file is: subsets\unpostponed.sub
    • Advanced - open the browser with topics that have recently been advanced (i.e. had their interval automatically shortened)
    • Previous - open the last but one browser

Other

[edit | edit source]
  • Other - open the submenu with more specialized browsers:
    • Leeches (Shift+F3) - open the browser with most difficult elements in the collection (note that filter parameters are already set in a way which makes finding difficult items easy)
    • Semi-leeches - open the browser with all elements that are leeches or will become leeches if they are forgotten yet once
    • Pending - open the browser with all pending elements, i.e. elements that are waiting for reading or memorization in the pending queue
    • Drill - open the browser with all elements scheduled for the final drill
    • Range - open the browser with all elements that fall into a selected range (e.g. elements with numbers from 1 to 1000)
    • History - open the browser with the list of elements that have recently been viewed in the element window
    • Branch - open the browser with all descendants of the currently selected branch in the Contents window (same as View : Branch (from the contents menu or Ctrl+Space in the Contents window or in the element window)

最近

[编辑|编辑源代码]
  • 最近- 打开浏览器与最近使用过的元素的子集
    • 学到了- 打开浏览器与元素最近使用于子集学习
    • 已审核- 打开浏览器与元素受当前子集审查（例如与查看全部）。因此你可以看到元素已经审核过的元素以及仍计划进一步审核的元素队列子集审查。的元素已经审查过的有其最后一次显示上次重复日期的字段（例如今天）。它们可以很容易地被过滤掉（例如，使用选择 ：选择浏览器范围于浏览器菜单）
    • 推迟- 打开浏览器与元素其中最近的推迟程序已执行。自动推迟不创建这个子集作为所有突出元素会自动推迟。不是全部元素在推迟子集实际上将被推迟（例如，由于没有满足推迟标准）。子集文件是：子集\postponing.sub
    • 推迟- 打开浏览器与元素谁的间隔最近已延长推迟。该子集可以手动创建推迟或与自动推迟。之后推迟该子集将包含推迟的元素并且将小于或等于推迟子集。看看哪个元素已自动推迟，请使用推迟。子集文件是：子集\postponed.sub
    • 不推迟- 打开浏览器与元素在最近的推迟操作中已被跳过。子集文件是：子集\unpostponed.sub
    • 高级- 打开浏览器与主题那些最近被先进的（即有他们的间隔自动缩短）
    • 上一页- 打开最后一个浏览器

其他

[编辑|编辑源代码]
  • 其他- 使用更专业的浏览器打开子菜单：
    • 水蛭（Shift+F3) - 打开浏览器中最困难的元素收藏（请注意，过滤器参数已经以某种方式设置，这使得查找变得困难项目容易）

    • 历史- 打开浏览器与清单元素最近在元素窗口
    • 分公司- 打开浏览器与所有后裔当前选定的分行在内容窗户（与查看 : 分行（来自内容菜单或Ctrl+空格键在内容窗户或在元素窗口）
