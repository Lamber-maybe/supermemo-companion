Importing PDF

[edit | edit source]

You can import PDF files into SuperMemo in three ways:

  • File : Import : Files and folders - Use this option for bulk import (e.g. large PDF archives).
  • Web import - Recommended when using predefined templates (make sure they include Article WV). Use Add predefined templates if it is missing, or create your own WebView template.
  • File : Import file - Available on the component menu in WebView component. Use this for single files stored on your disk

PDF component modes

[edit | edit source]

All PDF processing in SuperMemo 20 is handled within the WebView component. Depending on your workflow, you can use one of three modes:

  • Viewer mode

Default interface in display mode. Uses the Chromium engine for standard reading, navigation, and bookmarks.

  • PDF editor

Available in editing mode (Ctrl+E). Enables incremental learning features such as highlighting, read-points, and extracts.

  • PDF → HTML converter

Converts the document to plain HTML. Useful for the classic incremental reading workflow.

PDF editor

[edit | edit source]

In the PDF editor, your reading progress (scroll position) is tracked and saved automatically. You can also manually mark your position using Bookmark in the top toolbar.

Selecting text opens a floating toolbar with key learning tools:

  • Highlight

Marks the selected text in yellow. Highlights and progress are stored with your elements.

  • Extract → SM

Sends the selected text directly to your knowledge base, creating a child element under the current PDF topic.

  • Picture extract

Creates an image extract from clipboard content.

  • Snip and extract

Creates an image extract using the Windows Snipping Tool.

Each extracted element automatically includes a back-reference to the source PDF and the exact page number.

The viewer sidebar lists all extracts. Clicking an entry jumps directly to its original location in the document.

导入 PDF

[编辑|编辑源代码]

您可以通过三种方式将 PDF 文件导入 SuperMemo：

  • 文件：导入：文件和文件夹- 使用此选项进行批量导入（例如大型 PDF 档案）。
  • 网页导入- 建议使用预定义模板时（确保它们包括文章 WV）。使用添加预定义模板如果缺少，或者创建您自己的网页视图模板。
  • 文件：导入文件- 可在组件菜单在WebView组件。将此用于存储在磁盘上的单个文件

PDF 组件模式

[编辑|编辑源代码]

SuperMemo 20 中的所有 PDF 处理均在WebView组件。根据您的工作流程，您可以使用以下三种模式之一：

  • 查看器模式

默认界面在显示方式。使用 Chromium 引擎进行标准阅读、导航和书签。

  • PDF编辑器

可用于编辑模式（Ctrl+E）。启用渐进学习突出显示等功能，读取点, 和提取物。

  • PDF → HTML 转换器

将文档转换为纯 HTML。对经典很有用增量阅读工作流程。

PDF编辑器

[编辑|编辑源代码]

在 PDF 编辑器中，您的阅读进度（滚动位置）会自动跟踪并保存。您还可以使用手动标记您的位置书签在顶部工具栏中。

选择文本会打开一个浮动工具栏，其中包含主要学习工具：

  • 亮点

将选定的文本标记为黄色。亮点和进度都存储在您的元素。

  • 提取物 → SM

将选定的文本直接发送到您的知识库，创建一个子项元素在当前 PDF 主题下。

  • 图片摘录

从剪贴板内容创建图像提取。

  • 剪切并提取

使用 Windows 截图工具创建图像提取。

每个提取的元素都会自动包含对源 PDF 的反向引用和确切的页码。

查看器侧边栏列出了所有摘录。单击某个条目会直接跳转到其在文档中的原始位置。
